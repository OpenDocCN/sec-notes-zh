# 015：清晰简明的沟通

![](img/3d3c6863c0ddbed18d56bf20ca1b450a_0.png)

![](img/3d3c6863c0ddbed18d56bf20ca1b450a_0.png)

在本节课中，我们将学习如何与利益相关者进行清晰、简洁且有效的沟通。这是网络安全工作中至关重要的一环。

欢迎回来。上一节我们讨论了利益相关者及其在组织内扮演的重要安全角色。本节中，我们来看看你在与这些利益相关者沟通时所扮演的角色。

传达给利益相关者的信息通常是敏感的。例如，如果你通过电子邮件向利益相关者通报最近发生的一次安全漏洞，那么注意沟通的内容和对象就非常重要。

不同的利益相关者可能需要了解不同的问题。因此，你与他们的沟通需要清晰、简洁且有针对性。

网络安全是一个注重细节的职业，因此在发送信息时，你必须时刻注意细节。

利益相关者通常非常忙碌。你的沟通应当精准，避免不必要的技术术语，并具有明确的目的。你不应让他们去猜测你发送邮件的缘由或此事与他们有何关联。

为了做到这一点，你可以向你的经理或直属主管提问，以了解你所沟通的利益相关者需要知道什么。

你可能还记得，之前我们讨论过拥有安全思维模式意味着什么。这种思维模式的一部分，就是对你所保护的资产和数据提出疑问。

例如，你可以提问：**`日常工作中，最重要的保护数据是什么？`** 或者 **`哪种安全工具对于保护我们的数据和资产最为重要或有用？`**

拥有安全思维模式还意味着理解对利益相关者而言最重要的事情，这样你才知道应该与他们分享哪些信息。

有效的沟通只涉及向利益相关者传达最相关的信息。及时了解安全问题有助于利益相关者更有效地开展工作。你与利益相关者沟通的角色，就是帮助他们获取这些信息。

这再次证明了你在安全团队中的角色是多么重要。

接下来，我们将讨论与利益相关者沟通时最重要的信息类型。

本节课中，我们一起学习了与利益相关者进行清晰、简洁沟通的重要性，以及如何通过提问和关注细节来确保沟通的有效性。
