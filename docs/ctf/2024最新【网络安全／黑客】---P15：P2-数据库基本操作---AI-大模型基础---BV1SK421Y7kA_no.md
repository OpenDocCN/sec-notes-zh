## 数据库基本操作命令

以下是三个最基础的数据库操作命令。

*   **创建数据库**：`CREATE DATABASE database_name;`
*   **删除数据库**：`DROP DATABASE database_name;`
*   **选择数据库**：`USE database_name;`

选择数据库（`USE`）是必要的，因为一个数据库服务中可能包含多个不同的数据库，每个数据库存储不同的数据。例如，`information_schema`数据库存放系统元数据，而用户创建的数据库则存放业务数据。

## MySQL数据类型

创建数据库后，我们需要在其中创建表来存储数据，这就必须定义每个字段的数据类型。MySQL的数据类型主要分为三类。

### 数值类型
用于存储数字。常用类型是`INT`，它足以满足大多数场景的需求。
*   `TINYINT`：范围-128到127。
*   `SMALLINT`：范围-32768到32767。
*   `INT`：范围约-21亿到21亿。
*   `BIGINT`：更大范围的整数。

### 日期与时间类型
用于存储日期和时间信息。
*   `DATE`：存储日期，格式为‘YYYY-MM-DD’。
*   `TIME`：存储时间或持续时间。
*   `YEAR`：存储年份。
*   `DATETIME`：存储日期和时间组合。

### 字符串类型
用于存储文本信息。
*   `CHAR`：定长字符串，长度0-255字符。
*   `VARCHAR`：变长字符串，长度0-65535字符。
*   `TEXT`：用于存储长文本字符串。
*   `BLOB`：用于存储二进制字符串数据。

## 实战操作：使用PHPStudy操作数据库

讲完了理论知识，我们来进行实际操作。首先确保PHPStudy中的MySQL服务已经启动。

1.  **连接数据库**：打开PHPStudy，在“数据库工具”中选择一个客户端（如phpMyAdmin或SQL-Front）进行连接。
2.  **创建数据库（命令方式）**：
    在SQL命令窗口中输入 `CREATE DATABASE forest1;` 并执行，即可创建一个名为`forest1`的空数据库。
3.  **注意命令执行**：点击“运行”按钮通常会执行窗口中的所有命令。如果重复执行已存在的`CREATE DATABASE`命令，会导致错误。
4.  **删除数据库**：输入 `DROP DATABASE forest1;` 可以删除该数据库。
5.  **图形化操作**：大部分数据库客户端也支持图形化操作，例如右键点击数据库列表选择“新建数据库”。但**强烈建议初学者多使用命令方式**，这有助于深刻理解SQL语法，对后续学习SQL注入等安全技术至关重要。
