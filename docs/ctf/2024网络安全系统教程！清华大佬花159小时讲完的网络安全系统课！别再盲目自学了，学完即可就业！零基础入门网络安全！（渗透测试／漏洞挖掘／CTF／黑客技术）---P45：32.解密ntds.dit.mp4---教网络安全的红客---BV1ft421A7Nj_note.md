## 概述：解密 `ntds.dit` 文件

`ntds.dit` 文件是活动目录的数据库，存储了域内所有用户的账户信息，包括密码哈希。获取文件本身后，我们需要使用特定工具将其中的哈希值提取出来。以下是四种常用的解密方法。

![](img/c3ed5ea1c42488bc90f63cf09841f524_20.png)

## 方法一：使用 QuarkPwDump 工具

![](img/c3ed5ea1c42488bc90f63cf09841f524_22.png)

![](img/c3ed5ea1c42488bc90f63cf09841f524_24.png)

QuarkPwDump 是一个 Windows 用户凭据提取工具，可以抓取 Windows 平台下的多种用户数据。我们主要用它来提取 `ntds.dit` 文件中的哈希值。

![](img/c3ed5ea1c42488bc90f63cf09841f524_26.png)

以下是该工具的核心使用参数：
```
QuarkPwDump.exe --hash-dump --output hashes.txt --ntds-file ntds.dit
```
*   `--hash-dump`：指定导出域哈希。
*   `--output`：指定输出文件。
*   `--ntds-file`：指定 `ntds.dit` 文件路径。

![](img/c3ed5ea1c42488bc90f63cf09841f524_28.png)

![](img/c3ed5ea1c42488bc90f63cf09841f524_30.png)

**注意**：在使用该工具前，通常需要先使用系统自带的 `esentutl` 命令对导出的数据库文件进行修复。
```
esentutl /p /o ntds.dit
```

## 方法二：使用 secretsdump.py 脚本

![](img/c3ed5ea1c42488bc90f63cf09841f524_37.png)

![](img/c3ed5ea1c42488bc90f63cf09841f524_39.png)

secretsdump 是 Impacket 工具套件中的一个 Python 脚本，功能强大但速度相对较慢。它同样可以用于解密 `ntds.dit` 文件。

![](img/c3ed5ea1c42488bc90f63cf09841f524_41.png)

![](img/c3ed5ea1c42488bc90f63cf09841f524_43.png)

![](img/c3ed5ea1c42488bc90f63cf09841f524_45.png)

![](img/c3ed5ea1c42488bc90f63cf09841f524_46.png)

以下是解密 `ntds.dit` 的核心命令：
```
secretsdump.py -system SYSTEM -ntds ntds.dit LOCAL
```
*   `-system`：指定从域控中同时导出的 `SYSTEM` 注册表文件。
*   `-ntds`：指定 `ntds.dit` 文件。

执行后，该脚本会列出文件中的所有用户及其对应的 NTLM 哈希值。

![](img/c3ed5ea1c42488bc90f63cf09841f524_48.png)

## 方法三：使用 NTDSDumpEx 工具

NTDSDumpEx 是一个专门用于高效破解 `ntds.dit` 文件并提取信息的工具。

![](img/c3ed5ea1c42488bc90f63cf09841f524_50.png)

![](img/c3ed5ea1c42488bc90f63cf09841f524_52.png)

![](img/c3ed5ea1c42488bc90f63cf09841f524_54.png)

以下是该工具的核心使用命令：
```
NTDSDumpEx.exe -s SYSTEM -f ntds.dit -o hashes.txt
```
*   `-s`：指定 `SYSTEM` 文件。
*   `-f`：指定 `ntds.dit` 文件。
*   `-o`：指定输出文本文件。你也可以使用 `--csv` 参数将结果输出为 CSV 格式，便于用 Excel 查看。

Mimikatz 是最常用的凭据提取工具之一。与前三种“离线”解密方式不同，Mimikatz 可以利用域控制器的目录复制服务（DCSync）在线提取哈希，无需直接处理 `ntds.dit` 文件。

在 Cobalt Strike 等渗透框架中，可以集成并调用 Mimikatz 的功能。

![](img/c3ed5ea1c42488bc90f63cf09841f524_85.png)

![](img/c3ed5ea1c42488bc90f63cf09841f524_86.png)

以下是几个常用的 Mimikatz DCSync 命令：

![](img/c3ed5ea1c42488bc90f63cf09841f524_95.png)

![](img/c3ed5ea1c42488bc90f63cf09841f524_97.png)

![](img/c3ed5ea1c42488bc90f63cf09841f524_99.png)

**1. 获取域内所有用户的哈希值**
```
mimikatz # lsadump::dcsync /domain:mingy.com /all /csv
```
这条命令会获取指定域（如 mingy.com）中所有用户的哈希，并以 CSV 格式输出。

**2. 获取指定用户的详细信息（如 KRBTGT 账户）**
```
mimikatz # lsadump::dcsync /domain:mingy.com /user:krbtgt
```
`KRBTGT` 是域中一个特殊的账户，其密码哈希用于生成 Kerberos 票据。获取此哈希对后续的“黄金票据”攻击至关重要。

![](img/c3ed5ea1c42488bc90f63cf09841f524_101.png)

**3. 获取所有用户的详细信息**
```
mimikatz # lsadump::dcsync /domain:mingy.com /all
```
这条命令会输出域内所有用户的完整详细信息，包括哈希、最后登录时间等。

![](img/c3ed5ea1c42488bc90f63cf09841f524_103.png)

![](img/c3ed5ea1c42488bc90f63cf09841f524_105.png)

![](img/c3ed5ea1c42488bc90f63cf09841f524_107.png)

**重要提示**：要成功执行 DCSync 攻击，你需要已经获得一个域用户权限（通常是域管理员或具有复制权限的账户）。

## 哈希值的利用

![](img/c3ed5ea1c42488bc90f63cf09841f524_109.png)

获取到密码哈希后，主要有两种利用方式：
1.  **密码破解**：尝试对哈希进行破解，还原出明文密码。
2.  **哈希传递攻击**：在无法破解的情况下，直接使用哈希值进行横向移动，访问网络中的其他系统。

![](img/c3ed5ea1c42488bc90f63cf09841f524_111.png)

![](img/c3ed5ea1c42488bc90f63cf09841f524_112.png)

## 总结

![](img/c3ed5ea1c42488bc90f63cf09841f524_116.png)

![](img/c3ed5ea1c42488bc90f63cf09841f524_118.png)

本节课中我们一起学习了四种解密 `ntds.dit` 文件以提取域用户密码哈希的方法：
*   **QuarkPwDump**：一个专用的 Windows 凭据导出工具。
*   **secretsdump**：Impacket 套件中的离线解密脚本。
*   **NTDSDumpEx**：高效的专用 `ntds.dit` 提取工具。
*   **Mimikatz DCSync**：利用域协议在线提取哈希，是最直接的方法之一。

![](img/c3ed5ea1c42488bc90f63cf09841f524_120.png)

![](img/c3ed5ea1c42488bc90f63cf09841f524_122.png)
