# Android逆向-基础篇：P26：章节3-19-虚拟机与实体机

在本节课中，我们将对安卓逆向中常用的虚拟机和实体机进行总结与对比，帮助你选择最适合逆向分析的环境。

## 🖥️ 虚拟机总结

上一节我们介绍了逆向的基本环境，本节中我们来看看虚拟机的特点。虚拟机是在电脑上模拟安卓系统的软件。

以下是三种常见虚拟机的特点：

*   **Android Studio 自带模拟器**：速度最慢，运行最卡顿。一般不推荐使用。
*   **Genymotion**：速度相当快，运行非常稳定。但兼容性不好。根据经验，如果PC端安装了VMware，就无法安装Genymotion，因为Genymotion依赖于VirtualBox。
*   **国内模拟器（如雷电模拟器）**：比Genymotion更好一些。速度很快，运行稳定，并且提供了多开、模拟地理位置等实体机级别的功能。但它有一个最大缺点：安卓版本太低，通常只有7.x版本。

## 📱 实体机总结

了解了虚拟机的局限性后，我们来看看实体机。实体机是指真实的安卓手机。

我个人认为，对于安卓逆向而言，使用低版本安卓的虚拟机并不合适。它可能更适合游戏工作室用于多开或外挂开发。但进行逆向分析，实体机是更好的选择。因为它速度最快，运行最稳定。

以下是实体机的代表机型与优势：

*   **代表机型**：国内品牌如华为和小米（我个人使用小米），国外品牌如谷歌亲儿子系列（Pixel）。由于我个人没有使用过谷歌机型，我认为使用国内品牌就很好。
*   **核心优势**：
    1.  **性能**：最快、最稳定。
    2.  **兼容性**：某些应用（如微信）必须在实体机上运行，无法在虚拟机上使用。未来这类限制会越来越多。
    3.  **Root便利性**：小米手机进行Root操作是最容易的之一。

因此，我建议使用实体机进行安卓逆向，并且推荐使用小米手机。

## 🎯 课程总结

本节课中我们一起学习了安卓逆向环境的选择。我们对比了Android Studio模拟器、Genymotion和国内模拟器等虚拟机的优缺点，并分析了实体机在性能、兼容性和Root便利性上的显著优势。最终结论是，为了获得最佳体验和兼容性，推荐使用实体机（特别是小米手机）作为安卓逆向的分析环境。
