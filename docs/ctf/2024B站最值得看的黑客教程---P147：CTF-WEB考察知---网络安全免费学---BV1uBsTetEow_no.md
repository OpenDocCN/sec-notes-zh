## CTF的由来与概念

CTF（Capture The Flag，夺旗赛）是一种流行的网络安全竞赛形式。它起源于1996年的DEFCON全球黑客大会，旨在通过模拟真实的安全场景，让参赛者在解决各类安全挑战中学习和提升技能。

## CTF的比赛模式

CTF比赛主要有三种模式。解题模式（Jeopardy）是最常见的，参赛队伍通过解决网络安全、密码学、逆向工程等各类独立题目获取分数。攻防模式（Attack-Defense）中，每个队伍需要维护自己的服务器，同时攻击其他队伍的服务器来得分。混合模式则结合了前两者的特点。

## CTF-WEB主要考察内容

上一节我们介绍了CTF的基本概念与模式，本节中我们来看看CTF-WEB方向具体考察哪些技术点。外部方向是本次公开课的主题，主要考察在打比赛时涉及的知识。以下是十大主要的考点：

1.  **PHP弱类型**
2.  **SQL注入**
3.  文件上传
4.  文件包含
5.  XSS跨站脚本攻击
6.  反序列化漏洞

以上列举的PHP弱类型、SQL注入、文件上传、文件包含、跨站脚本攻击、反序列化等，都是我们CTF-WEB方向主要的考察方向。
