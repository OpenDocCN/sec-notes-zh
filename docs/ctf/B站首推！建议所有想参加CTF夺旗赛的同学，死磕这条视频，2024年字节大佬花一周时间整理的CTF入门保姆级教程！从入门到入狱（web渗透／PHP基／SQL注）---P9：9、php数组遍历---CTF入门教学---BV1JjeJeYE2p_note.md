# CTF入门教学：P9：PHP数组遍历 🚗

![](img/2a743c42ed74b0f053f05f674934d208_1.png)

在本节课中，我们将要学习PHP中一个非常重要的概念——数组。数组允许我们在单个变量中存储多个值，这在处理列表数据（如一系列车名）时非常有用。我们将学习如何创建数组、访问数组中的值，以及如何遍历数组中的所有元素。

![](img/2a743c42ed74b0f053f05f674934d208_2.png)

## 什么是数组？

![](img/2a743c42ed74b0f053f05f674934d208_4.png)

![](img/2a743c42ed74b0f053f05f674934d208_6.png)

数组是一种特殊的变量，它可以在单个变量中存储多个值。

在编写代码时，通常我们会将一个值赋给一个变量。但使用数组，我们可以用一个变量来存储多个值。

![](img/2a743c42ed74b0f053f05f674934d208_8.png)

## 如何创建和访问数组？

![](img/2a743c42ed74b0f053f05f674934d208_9.png)

![](img/2a743c42ed74b0f053f05f674934d208_10.png)

以下是创建数组的方法。我们使用 `array()` 函数，并在括号内写入多个值，值之间用逗号分隔。

```php
$cars = array("Volvo", "BMW", "Toyota");
```

![](img/2a743c42ed74b0f053f05f674934d208_12.png)

字符串值需要用引号引起来，而数字则不需要。

![](img/2a743c42ed74b0f053f05f674934d208_14.png)

![](img/2a743c42ed74b0f053f05f674934d208_16.png)

要访问数组中的值，我们需要使用下标（索引）。数组的下标从0开始。

![](img/2a743c42ed74b0f053f05f674934d208_18.png)

```php
echo $cars[0] . ", " . $cars[1] . ", " . $cars[2];
```

这段代码将输出：`Volvo, BMW, Toyota`。我们通过变量名加方括号和下标的方式来访问特定位置的值。点号 `.` 是之前学过的字符串连接运算符。

## 为什么需要数组？

假设你需要存储一系列车名。如果只有三辆车，为每辆车定义一个单独的变量或许可以接受。但如果你需要存储300辆、500辆甚至1000辆车的名字，为每一辆都定义一个变量将非常低效且难以管理。

![](img/2a743c42ed74b0f053f05f674934d208_20.png)

![](img/2a743c42ed74b0f053f05f674934d208_21.png)

![](img/2a743c42ed74b0f053f05f674934d208_22.png)

解决方案就是使用数组。我们可以将所有值放入一个数组中，然后通过键（下标）来访问它们。数组本质上是一个“键值对”的集合。

## 数组的三种类型

在PHP中，使用 `array()` 函数创建数组。数组主要有三种类型：
1.  **数值数组**：带有数字ID键的数组。
2.  **关联数组**：带有指定键的数组，每个键关联一个值。
3.  **多维数组**：包含一个或多个数组的数组。

![](img/2a743c42ed74b0f053f05f674934d208_24.png)

本节课我们主要关注数值数组。数值数组会自动分配数字ID键（下标），从0开始。

你也可以手动分配下标：

```php
$cars = array();
$cars[0] = "Volvo";
$cars[1] = "BMW";
$cars[2] = "Toyota";
```

![](img/2a743c42ed74b0f053f05f674934d208_26.png)

![](img/2a743c42ed74b0f053f05f674934d208_27.png)

![](img/2a743c42ed74b0f053f05f674934d208_28.png)

甚至可以不按顺序赋值：

```php
$cars = array();
$cars[0] = "Volvo";
$cars[2] = "BMW"; // 跳过了下标1
```

第一种方式是定义时直接赋值，第二种方式是先定义空数组再单独赋值。

![](img/2a743c42ed74b0f053f05f674934d208_30.png)

![](img/2a743c42ed74b0f053f05f674934d208_31.png)

![](img/2a743c42ed74b0f053f05f674934d208_32.png)

## 获取数组的长度

![](img/2a743c42ed74b0f053f05f674934d208_33.png)

当数组中有很多元素时，手动去数个数是不现实的。PHP提供了 `count()` 函数来获取数组的长度（即元素个数）。

```php
$cars = array("Volvo", "BMW");
echo count($cars);
```

![](img/2a743c42ed74b0f053f05f674934d208_35.png)

![](img/2a743c42ed74b0f053f05f674934d208_36.png)

这段代码将输出 `2`，表示 `$cars` 数组中有两个元素。

![](img/2a743c42ed74b0f053f05f674934d208_38.png)

## 遍历数组

上一节我们介绍了如何获取数组的长度，本节中我们来看看如何遍历数组。遍历数组意味着逐个访问并处理数组中的每一个元素。

![](img/2a743c42ed74b0f053f05f674934d208_40.png)

![](img/2a743c42ed74b0f053f05f674934d208_41.png)

我们可以使用 `for` 循环来实现数组的遍历。思路是：从下标0开始，循环到数组长度减1的位置，在循环体内通过动态的下标来访问每一个元素。

![](img/2a743c42ed74b0f053f05f674934d208_42.png)

```php
$cars = array("Volvo", "BMW", "Toyota");
$arrlength = count($cars);

![](img/2a743c42ed74b0f053f05f674934d208_44.png)

for($i = 0; $i < $arrlength; $i++) {
    echo $cars[$i];
    echo "<br>";
}
```

![](img/2a743c42ed74b0f053f05f674934d208_46.png)

**代码解析**：
1.  `$arrlength = count($cars);` 获取数组长度，这里是3。
2.  `for($i = 0; $i < $arrlength; $i++)` 设置循环。注意循环条件是 `$i < $arrlength`，而不是 `$i <= $arrlength`。因为下标从0开始，最大有效下标是2（长度3-1）。如果使用 `<=`，`$i` 会等于3，访问 `$cars[3]` 会导致错误。
3.  `echo $cars[$i];` 在每次循环中，`$i` 的值分别为0, 1, 2，因此会依次输出 `$cars[0]`, `$cars[1]`, `$cars[2]` 的值。
4.  `echo “<br>”;` 用于在每次输出后换行，使结果更清晰。

![](img/2a743c42ed74b0f053f05f674934d208_47.png)

![](img/2a743c42ed74b0f053f05f674934d208_49.png)

运行这段代码，将会输出：
```
Volvo
BMW
Toyota
```

除了 `for` 循环，PHP还提供了更简洁的 `foreach` 循环来遍历数组，这在后续课程中会学到。
