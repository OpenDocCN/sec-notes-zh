## 数据安全风险概述

数据泄露、数据损坏、数据加密都属于数据安全的风险范畴。

我们经常会在网上看到某某企业被勒索的新闻。例如，以前在B站上就有很多UP主提到自己的服务器中了勒索病毒。

这导致他们的素材全部丢失，造成了损失。这就是一个典型的数据泄露以及数据损坏事件。

其次，一些数据在数据库中没有进行严格的加密，并且没有对身份进行严格的验证，从而被黑客恶意获取。

这些数据可能包括身份证号、姓名以及收获地址等敏感信息。

## 敏感信息的危害

对于普通人来说，这些敏感信息看起来似乎没有直接用处。很多人认为现在上网如同“裸奔”，姓名等信息似乎人尽皆知。

但是，请大家思考一下。

如果这些敏感信息被电信诈骗等违法分子大量收集起来。

那么，他们在实施犯罪行为时，是不是就更加便利，成功的可能性也更高了呢？因此，数据安全尤为重要。

## 关于“熊猫烧香”的辨析

有同学提到了“熊猫烧香”病毒。

“熊猫烧香”其实不属于数据加密攻击，它主要是破坏了操作系统，属于数据损坏的范畴。

它的作用是把你的操作系统破坏掉。

但需要注意一点，通常情况下，红队（攻击方）的目的不是破坏你的电脑。

红队更想做到的是控制你的电脑，是控制你，而不是破坏你。

如果一个红队给你植入了类似“熊猫烧香”的破坏性病毒，导致你的电脑无法启动。

那么这就变成了一次破坏性的、针对操作系统的攻击。但这通常不是红队的目的，他们更想控制你，观察你的行为。

例如，他们可能想查看你的手机摄像头，这才是红队更常做的事情。
