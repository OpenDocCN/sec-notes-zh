*   预装了丰富的安全工具，例如Nmap、Metasploit、Burp Suite等。

## 📝 面试常见Linux考点

了解Linux的重要性后，本节我们将聚焦于面试中可能被问及的Linux核心知识点。掌握这些内容有助于在技术面试中展现扎实的基础。

以下是面试中常见的Linux相关主题列表：
*   **基础命令**：文件操作（`ls`, `cd`, `cp`, `mv`, `rm`）、文本处理（`grep`, `awk`, `sed`）、权限管理（`chmod`, `chown`）。
*   **系统管理**：进程查看与管理（`ps`, `top`, `kill`）、网络配置与诊断（`ifconfig`, `netstat`, `ping`）、服务管理（`systemctl`）。
*   **安全相关**：用户与组管理、日志分析（`/var/log/`目录）、防火墙配置（`iptables`/`firewalld`）。
*   **Shell脚本**：基本的Bash脚本编写能力，用于自动化简单任务。
*   **环境与配置**：环境变量（`$PATH`）、文件系统结构、软件包管理（`apt`, `yum`）。

## 🔧 核心概念示例

为了更具体地理解，我们来看两个核心概念的表示方式。

**1. 文件权限概念**
Linux文件权限通常用一串字符表示，例如 `-rwxr-xr--`。这可以转化为数字权限码。其对应关系由公式表示：
> 权限数字 = 所有者权限×100 + 所属组权限×10 + 其他用户权限×1
> 其中，r（读）=4，w（写）=2，x（执行）=1。

因此，`rwxr-xr--` 对应的数字权限计算为：
所有者：rwx = 4+2+1 = 7
所属组：r-x = 4+0+1 = 5
其他用户：r-- = 4+0+0 = 4
最终权限码为 **754**。

**2. 常用命令代码片段**
查找当前目录下所有包含“password”字符串的.txt文件，可以使用`grep`命令：
```bash
grep -r "password" *.txt
```
这条命令中，`-r`参数表示递归搜索，`"password"`是搜索模式，`*.txt`指定了目标文件类型。
