# CTF入门指南：P146：竞赛内容概述 🎯

在本节课中，我们将要学习CTF（Capture The Flag）网络安全竞赛的核心内容构成。CTF比赛通常涵盖多个技术方向，了解这些方向有助于初学者规划学习路径。

## 六大核心方向 🧩

上一节我们介绍了CTF竞赛的整体概念，本节中我们来看看其具体的六大技术方向。这些方向构成了CTF题目的主要类型。

以下是CTF竞赛的六大核心方向：

1.  **Web方向**  
    本课程为CTF Web方向的公开课，主要讲解Web安全相关的题目。Web方向涉及网站、Web应用的安全漏洞挖掘与利用。

2.  **逆向工程方向**  
    逆向工程主要分析二进制程序，理解其逻辑并找出隐藏信息或漏洞。

3.  **Pwn方向**  
    Pwn属于二进制漏洞利用范畴，涉及程序内存破坏、漏洞利用等技术。

4.  **密码学方向**  
    密码学方向涵盖加解密技术，涉及各类密码算法（如RSA、AES等）的分析与破解。

5.  **移动安全方向**  
    移动安全主要针对安卓或IOS系统，涉及移动应用逆向分析、安装包漏洞挖掘等。

6.  **Misc方向**  
    Misc意为安全杂项，包含无法归类到上述方向但与信息安全相关的题目，例如图片隐写、数字取证等。
