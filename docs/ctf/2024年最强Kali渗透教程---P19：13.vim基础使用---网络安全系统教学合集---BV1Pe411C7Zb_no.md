# Kali Linux渗透测试：P19：Vim编辑器基础使用 🖥️

在本节课中，我们将学习Vim编辑器的基础操作。Vim是Linux系统中一个功能强大的文本编辑器，尤其在命令行环境下进行文件编辑和代码审计时不可或缺。掌握Vim的基本使用，是进行网络安全工作的基础技能之一。

## 概述

当我们需要编辑一个软件或文件时，可以使用图形化编辑器如Gedit。然而，在没有图形界面的环境中，我们只能通过Vim这类命令行编辑器进行操作。Vim有两种常见形式：`vi`和`vim`，两者的使用方式基本相同。

## Vim的基本使用与模式

![](img/851ed9850218bcffabe302638417bfca_4.png)

Vim编辑器主要包含三种工作模式：**命令模式**、**输入模式**和**底线命令模式**。理解这三种模式是熟练使用Vim的关键。

![](img/851ed9850218bcffabe302638417bfca_6.png)

![](img/851ed9850218bcffabe302638417bfca_7.png)

### 1. 命令模式

![](img/851ed9850218bcffabe302638417bfca_9.png)

![](img/851ed9850218bcffabe302638417bfca_10.png)

启动Vim后，默认进入命令模式。在此模式下，无法直接输入文本，但可以执行各种编辑命令。

以下是命令模式下的一些常用操作：

![](img/851ed9850218bcffabe302638417bfca_12.png)

![](img/851ed9850218bcffabe302638417bfca_13.png)

*   **`i`**：在当前光标位置前进入输入模式。
*   **`a`**：在当前光标位置后进入输入模式。
*   **`o`**：在当前行下方新建一行并进入输入模式。
*   **`dd`**：删除当前行。
*   **`ndd`**：删除当前行及其后的n-1行（例如，`5dd`删除5行）。
*   **`yy`**：复制当前行。
*   **`nyy`**：复制当前行及其后的n-1行。
*   **`p`**：粘贴已复制或删除的内容。
*   **`u`**：撤销上一次操作。
*   **`Ctrl + f`**：向下翻页。
*   **`Ctrl + b`**：向上翻页。
*   **`ESC`**：从其他模式退回到命令模式。

![](img/851ed9850218bcffabe302638417bfca_15.png)

![](img/851ed9850218bcffabe302638417bfca_16.png)

![](img/851ed9850218bcffabe302638417bfca_18.png)

### 2. 输入模式

在命令模式下按下 `i`、`a`、`o` 等键即可进入输入模式。此时，屏幕底部通常会显示 `-- INSERT --` 或类似提示，表示可以自由输入和编辑文本内容。

![](img/851ed9850218bcffabe302638417bfca_20.png)

![](img/851ed9850218bcffabe302638417bfca_21.png)

![](img/851ed9850218bcffabe302638417bfca_22.png)

### 3. 底线命令模式

![](img/851ed9850218bcffabe302638417bfca_23.png)

![](img/851ed9850218bcffabe302638417bfca_24.png)

在命令模式下，按下冒号 `:` 即可进入底线命令模式。光标会移动到屏幕底部，可以输入各种保存、退出或设置命令。

![](img/851ed9850218bcffabe302638417bfca_26.png)

以下是底线命令模式下的一些常用操作：

![](img/851ed9850218bcffabe302638417bfca_28.png)

*   **`:w`**：保存文件。
*   **`:w!`**：强制保存文件。
*   **`:q`**：退出Vim（如果文件有修改且未保存，会提示错误）。
*   **`:q!`**：强制退出Vim，不保存任何修改。
*   **`:wq`** 或 **`:x`**：保存文件并退出。
*   **`:wq!`**：强制保存文件并退出。
*   **`:set nu`**：显示行号。
*   **`:set nonu`**：隐藏行号。
*   **`/关键词`**：在文件中向下搜索指定关键词。
*   **`?关键词`**：在文件中向上搜索指定关键词。

## 模式切换流程

Vim三种模式之间的切换遵循一个清晰的流程，理解这个流程是高效使用Vim的基础。

1.  使用 `vim 文件名` 命令打开文件，自动进入**命令模式**。
2.  在命令模式下，按 `i`、`a`、`o` 等键进入**输入模式**，开始编辑文本。
3.  在输入模式下，按 `ESC` 键返回**命令模式**。
4.  在命令模式下，按 `:` 键进入**底线命令模式**，执行保存、退出等操作。
5.  在底线命令模式下，按 `ESC` 键可取消当前命令并返回命令模式。

![](img/851ed9850218bcffabe302638417bfca_30.png)

**核心流程可以简化为：启动（命令模式） -> 按 `i`（输入模式） -> 编辑 -> 按 `ESC`（命令模式） -> 按 `:`（底线命令模式） -> 输入 `wq`（保存并退出）。**

## 实战演示

![](img/851ed9850218bcffabe302638417bfca_32.png)

上一节我们介绍了Vim的三种模式及其切换，本节我们通过一个简单的例子来实际操作。

1.  **打开文件**：在终端输入 `vim test.txt` 打开（或创建）一个文件。
2.  **进入输入模式**：按下 `i` 键，屏幕底部出现 `-- INSERT --`，进入输入模式。
3.  **编辑内容**：输入一些文本，例如 `Hello, Kali Linux.`。
4.  **返回命令模式**：按下 `ESC` 键，退出输入模式。
5.  **保存并退出**：输入 `:wq` 然后按回车，文件被保存并退出Vim。
