## Kali Linux的安装方式

![](img/61417ab5981420ae052143983ced4d95_62.png)

![](img/61417ab5981420ae052143983ced4d95_64.png)

在介绍完什么是Kali Linux之后，我们一起来了解一下它的安装。首先，我们需要下载Kali Linux系统。

![](img/61417ab5981420ae052143983ced4d95_68.png)

![](img/61417ab5981420ae052143983ced4d95_70.png)

其官方下载网址是：`https://www.kali.org/get-kali/`。我们直接搜索“Kali”即可找到。

![](img/61417ab5981420ae052143983ced4d95_72.png)

打开官网后，可以看到这里提供了一些可供下载的版本。新版本不像之前有那么多针对特定桌面环境（如GNOME、KDE等）的镜像。

![](img/61417ab5981420ae052143983ced4d95_74.png)

![](img/61417ab5981420ae052143983ced4d95_76.png)

![](img/61417ab5981420ae052143983ced4d95_77.png)

官网主要提供以下几种版本：
*   **Kali Linux 64-bit Installer**: 标准的64位安装版本。
*   **Kali Linux 32-bit Installer**: 标准的32位安装版本。
*   **Kali Linux Live**: 实时运行版本，可以放到U盘里直接运行。
*   **Kali Linux NetInstaller**: 在线安装版本，镜像文件最小，大部分工具需要在线下载安装。

## 使用虚拟机版本快速体验

![](img/61417ab5981420ae052143983ced4d95_109.png)

![](img/61417ab5981420ae052143983ced4d95_110.png)

上一节我们介绍了Kali Linux的几种安装方式，本节中我们来看看如何使用预配置的虚拟机版本快速启动Kali。

使用虚拟机版本非常简单。我们直接打开下载好的虚拟机文件（例如`.vmx`文件），然后在VMware中开启此虚拟机即可。

开启虚拟机会进入一个启动界面，我们选择第一个选项“Live system (amd64)”以图形化界面启动。

启动完毕后，需要进行登录。我们可以尝试使用`root`用户登录，但在新版本中这会失败。默认需要使用`kali`用户登录，密码也是`kali`。

登录成功后，界面如下图所示。如果之前使用过`root`用户登录，主机名会显示为红色。而默认使用`kali`用户登录，主机名显示为白色。

![](img/61417ab5981420ae052143983ced4d95_112.png)

![](img/61417ab5981420ae052143983ced4d95_114.png)

![](img/61417ab5981420ae052143983ced4d95_32.png)

![](img/61417ab5981420ae052143983ced4d95_116.png)

使用`kali`这种低权限用户进行操作（例如安装软件）时，通常需要加上`sudo`命令来提权，比较繁琐。如果我们对系统了解，可以直接启用并使用`root`用户。

以下是启用`root`用户并设置密码的步骤：
1.  首先切换到`root`用户或使用`sudo`提权。
2.  执行命令 `sudo passwd root`。
3.  输入当前`kali`用户的密码进行验证。
4.  输入新的`root`用户密码并确认。

![](img/61417ab5981420ae052143983ced4d95_118.png)

![](img/61417ab5981420ae052143983ced4d95_120.png)

![](img/61417ab5981420ae052143983ced4d95_122.png)

密码修改成功后，就可以使用`su root`命令切换到`root`用户，或者直接用`root`用户登录了。

进入系统后，我们可以大致了解一下如何使用。点击左上角的“应用程序”菜单，可以看到工具被分门别类地整理好了。

![](img/61417ab5981420ae052143983ced4d95_124.png)

例如：
*   **信息收集工具**：包括DNS分析、IDS检测、存活主机探测、端口扫描等。
*   **漏洞分析工具**：包含各种漏洞扫描和利用工具。

![](img/61417ab5981420ae052143983ced4d95_126.png)

这些分类使得工具查找和使用更加方便。

我们最常用的可能是终端（Terminal）。如果你对Linux系统操作不熟悉，建议课后加强学习，这对于后续从事Web安全和渗透测试学习至关重要。

![](img/61417ab5981420ae052143983ced4d95_128.png)

系统语言可以改成中文，但熟悉英文界面更有助于学习。中文输入法可以通过安装相关软件包（如`fcitx`、`ibus-pinyin`）来添加。

![](img/61417ab5981420ae052143983ced4d95_130.png)

## 使用ISO镜像安装Kali Linux

![](img/61417ab5981420ae052143983ced4d95_132.png)

上一节我们体验了虚拟机版本的便捷，本节中我们来看看如何使用ISO镜像文件进行更自定义的安装。

![](img/61417ab5981420ae052143983ced4d95_134.png)

![](img/61417ab5981420ae052143983ced4d95_136.png)

首先，你需要准备好Kali Linux的ISO镜像文件。然后，在VMware中新建一个虚拟机。

在新建虚拟机时，建议选择“自定义（高级）”模式，以便进行更详细的配置。

![](img/61417ab5981420ae052143983ced4d95_138.png)

在安装来源步骤，选择“稍后安装操作系统”。在操作系统类型中，选择“Linux”，版本选择“Debian 10.x 64位”。

接下来，为虚拟机命名并选择安装位置。**不建议安装在C盘**，因为虚拟机文件会占用较大空间。

![](img/61417ab5981420ae052143983ced4d95_140.png)

根据自己电脑的配置，设置处理器的数量和核心数，以及分配的内存大小（例如，电脑内存8G，可分配2G给虚拟机）。

![](img/61417ab5981420ae052143983ced4d95_142.png)

**关键步骤：网络类型选择**
VMware提供了几种网络连接类型，理解它们对后续使用很重要：
*   **桥接模式 (Bridged)**：虚拟机的网卡通过虚拟桥接直接连接到物理网络。虚拟机会获得一个与物理机同网段的IP地址，像一台独立的物理设备一样存在于网络中。**会消耗局域网内的IP地址**。
*   **NAT模式**：虚拟机通过一个虚拟网卡（如VMnet8）与主机通信，并通过主机的IP地址访问外网。虚拟机的IP地址通常在一个独立的私有网段（如`192.168.x.x`），**不会消耗物理网络的IP资源**。
*   **仅主机模式 (Host-Only)**：虚拟机只与主机及其他同模式虚拟机通信，**无法访问外网**。

![](img/61417ab5981420ae052143983ced4d95_144.png)

对于初学者，选择**桥接模式**或**NAT模式**均可。桥接模式更方便物理机直接访问虚拟机。这里我们选择桥接模式。

![](img/61417ab5981420ae052143983ced4d95_146.png)

![](img/61417ab5981420ae052143983ced4d95_147.png)

![](img/61417ab5981420ae052143983ced4d95_149.png)

![](img/61417ab5981420ae052143983ced4d95_151.png)

![](img/61417ab5981420ae052143983ced4d95_153.png)

![](img/61417ab5981420ae052143983ced4d95_155.png)

![](img/61417ab5981420ae052143983ced4d95_157.png)

接下来配置磁盘。建议磁盘大小设置大一些（如40GB或以上），以便有足够空间安装工具和存储数据。选择“将虚拟磁盘存储为单个文件”。

创建完成后，在虚拟机设置中，选择我们下载好的Kali Linux ISO镜像文件作为CD/DVD驱动器。

![](img/61417ab5981420ae052143983ced4d95_159.png)

![](img/61417ab5981420ae052143983ced4d95_161.png)

![](img/61417ab5981420ae052143983ced4d95_162.png)

![](img/61417ab5981420ae052143983ced4d95_164.png)

启动虚拟机，会进入安装引导界面。选择“Graphical install”（图形化安装）。

![](img/61417ab5981420ae052143983ced4d95_166.png)

安装过程会提示选择语言、区域、键盘布局等。建议选择“中文（简体）”。

![](img/61417ab5981420ae052143983ced4d95_168.png)

![](img/61417ab5981420ae052143983ced4d95_169.png)

然后设置主机名（如`kali`），域名（本地使用可留空）。

![](img/61417ab5981420ae052143983ced4d95_171.png)

![](img/61417ab5981420ae052143983ced4d95_173.png)

![](img/61417ab5981420ae052143983ced4d95_174.png)

![](img/61417ab5981420ae052143983ced4d95_176.png)

接着创建一个普通用户账户（用于替代root执行非管理任务），输入用户名、密码等信息。**注意**：在服务器或公网环境，请务必使用强密码。

![](img/61417ab5981420ae052143983ced4d95_178.png)

![](img/61417ab5981420ae052143983ced4d95_180.png)

![](img/61417ab5981420ae052143983ced4d95_182.png)

安装程序会进行磁盘分区。对于新手，可以选择“使用整个磁盘”并“将所有文件放在同一个分区中”，让安装程序自动完成分区。

如果对Linux文件系统熟悉，可以选择“手动”分区。常见的分区方案涉及以下几个挂载点：
*   **`/boot`**：系统启动引导文件，一般分配200MB-400MB。
*   **`/swap`**：交换空间，用于系统内存不足时使用，一般分配2GB-4GB。
*   **`/home`**：用户家目录，如果系统有多个用户，建议分配较大空间。
*   **`/`**：根目录，包含系统所有其他目录，分配剩余大部分空间。

选择“结束分区设定并将修改写入磁盘”，确认后安装程序开始复制系统文件。

安装过程中，可能会提示配置HTTP代理（如果网络需要），没有则留空。

![](img/61417ab5981420ae052143983ced4d95_184.png)

![](img/61417ab5981420ae052143983ced4d95_186.png)

![](img/61417ab5981420ae052143983ced4d95_188.png)

系统安装完成后，会提示重启。重启后即可使用之前设置的用户名和密码登录。

## 安装后的网络配置与问题排查

![](img/61417ab5981420ae052143983ced4d95_199.png)

![](img/61417ab5981420ae052143983ced4d95_200.png)

![](img/61417ab5981420ae052143983ced4d95_202.png)

![](img/61417ab5981420ae052143983ced4d95_204.png)

![](img/61417ab5981420ae052143983ced4d95_206.png)

系统安装完成后，可能会遇到无法上网的问题。这通常与之前选择的网络模式配置有关。

![](img/61417ab5981420ae052143983ced4d95_208.png)

![](img/61417ab5981420ae052143983ced4d95_210.png)

![](img/61417ab5981420ae052143983ced4d95_212.png)

如果你在创建虚拟机时选择了**桥接模式**，但无法上网，可能需要检查VMware的虚拟网络编辑器，确保桥接到了正确的物理网卡（有线网卡或无线网卡）。

![](img/61417ab5981420ae052143983ced4d95_214.png)

![](img/61417ab5981420ae052143983ced4d95_216.png)

![](img/61417ab5981420ae052143983ced4d95_218.png)

如果你选择了**NAT模式**，虚拟机通常会通过DHCP自动获取IP，一般可以直接上网。

![](img/61417ab5981420ae052143983ced4d95_220.png)

![](img/61417ab5981420ae052143983ced4d95_222.png)

![](img/61417ab5981420ae052143983ced4d95_224.png)

![](img/61417ab5981420ae052143983ced4d95_226.png)

![](img/61417ab5981420ae052143983ced4d95_227.png)

![](img/61417ab5981420ae052143983ced4d95_229.png)

![](img/61417ab5981420ae052143983ced4d95_231.png)

配置完成后，在Kali Linux终端中，可以尝试`ping`一个外网地址（如`8.8.8.8`）测试网络连通性。

![](img/61417ab5981420ae052143983ced4d95_232.png)

![](img/61417ab5981420ae052143983ced4d95_234.png)

![](img/61417ab5981420ae052143983ced4d95_236.png)

![](img/61417ab5981420ae052143983ced4d95_238.png)

![](img/61417ab5981420ae052143983ced4d95_240.png)

![](img/61417ab5981420ae052143983ced4d95_242.png)

![](img/61417ab5981420ae052143983ced4d95_244.png)

如果`ping`不通IP但可以`ping`通IP地址，可能是DNS解析问题。可以编辑`/etc/resolv.conf`文件，添加公共DNS服务器，例如：
```
nameserver 8.8.8.8
nameserver 114.114.114.114
```

![](img/61417ab5981420ae052143983ced4d95_246.png)

如果需要更改主机名，可以编辑`/etc/hostname`文件。
