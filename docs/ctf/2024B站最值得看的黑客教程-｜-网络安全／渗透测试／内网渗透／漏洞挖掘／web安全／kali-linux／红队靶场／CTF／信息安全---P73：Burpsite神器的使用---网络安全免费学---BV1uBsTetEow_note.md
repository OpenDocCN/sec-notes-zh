# 网络安全入门：P73：Burpsuite神器的使用 🔧

## 工具概述与重要性

首先，我们去了解对应的一个工具，也就是挖掘漏洞会用到的工具。Burpsuite是我们去挖掘Web漏洞必备的工具，基本上90%的漏洞都可以通过Burpsuite去进行发现和测试。
