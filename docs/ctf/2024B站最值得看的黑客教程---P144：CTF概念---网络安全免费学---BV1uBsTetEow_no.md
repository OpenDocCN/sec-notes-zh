# CTF入门教程：P144：CTF概念

## 概述
在本节课中，我们将要学习CTF（夺旗赛）的基本概念。我们将了解CTF的含义、起源、发展以及它在网络安全领域中的重要性。通过本节内容，你将建立起对CTF竞赛的初步认识。

## 认识CTF

上一节我们介绍了课程的整体框架，本节中我们来看看什么是CTF。

CTF是capture the flag的缩写。CTF是这三个单词的首字母缩写。中文称之为夺旗赛，意指抢夺旗帜flag。

在解题过程中，目标就是找到flag。拿到flag意味着题目完成。提交flag后即可得分。

## CTF的起源与发展

了解了CTF的基本含义后，我们来看看它的历史。

CTF起源于1996年的DEFCON全球黑客大会。当时是计算机网络刚刚兴起的年代，会上网的人还不多。一群上网的黑客在论坛上相约进行聚会。

在聚会过程中，他们想切磋彼此的计算机能力。后来逐渐演化成CTF比赛。CTF比赛是切磋计算机能力的一种模式。它通过比赛考察参与者对安全的理解。

现在CTF比赛越来越流行。这是一种很好的培养和考察网络安全能力的比赛形式。因此它现在越来越火。国内也有很多CTF比赛，基本上每个月都有一个比较大型的CTF比赛。

DEFCON CTF是目前全球技术水平和影响力最高的CTF竞赛，因为它是起源。以上就是CTF的概念。

## 总结
本节课中我们一起学习了CTF的基本概念。我们了解到CTF是“夺旗赛”的缩写，其核心目标是找到并提交**flag**。它起源于1996年的DEFCON大会，并已发展成为考察和培养网络安全技能的重要竞赛形式。现在，CTF在国内外都非常流行。
