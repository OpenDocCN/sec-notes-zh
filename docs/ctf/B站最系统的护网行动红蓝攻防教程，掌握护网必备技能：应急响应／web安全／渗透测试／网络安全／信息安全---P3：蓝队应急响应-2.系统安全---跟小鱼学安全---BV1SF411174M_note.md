# 护网行动红蓝攻防教程：P3：蓝队应急响应-2.系统安全 🔐

在本节课中，我们将要学习蓝队应急响应中关于系统安全的核心知识。系统安全是防御红队攻击的关键环节，涉及对计算机、服务器乃至移动设备的全面防护。我们将探讨红队常见的系统攻击手段，并了解蓝队应如何构建防御体系。

上一节我们介绍了应急响应的基础概念，本节中我们来看看系统安全的具体内涵。

系统安全涵盖的范围很广，不仅包括我们日常使用的办公电脑，也包含手机等移动设备。红队的一大技术方向，就是实现对目标设备的隐秘控制。

一个典型的高级红队技术是**安卓手机远控**。其核心目标是在控制目标手机时，不让机主察觉，实现无感化操作，例如悄无声息地调用摄像头拍照，或持续监控手机活动。掌握此类技术通常是中级或高级红队的标志。

那么，系统安全具体需要防范哪些威胁呢？主要包括各类恶意软件。

以下是常见的系统安全威胁类型：
*   **手机病毒**：针对移动设备的恶意软件。
*   **勒索病毒**：加密用户文件并索要赎金的软件。
*   **挖矿木马**：在受害者计算机上隐秘运行，消耗资源进行加密货币挖矿的程序。
*   **远程后门**：攻击者植入的，用于维持长期隐蔽访问的通道。

除了上述基于网络的攻击，红队还有一种非常特殊的攻击方式，即近源渗透。

近源渗透，或称物理渗透，是指攻击者通过物理接近目标来实施攻击。例如，攻击企业内部的无线网络，或直接在目标电脑上插入携带病毒的U盘。多年前的明星隐私泄露事件，很大程度上就是因为在送修电脑时，硬盘数据被非法读取，这正是一种硬件安全失守的案例。

因此，许多红队会尝试潜伏到目标企业内部进行此类近源渗透。这意味着蓝队的防御职责非常广泛。

蓝队不仅要防御网站和应用层面的攻击，还必须严防红队通过物理方式潜入内部网络。在护网行动中，成功发现并阻止此类渗透行为会为蓝队加分。

最后，必须郑重强调：**未经授权的任何渗透测试，包括近源渗透，都是违法行为**。正规的红队在执行线下物理渗透任务前，必须获得明确的书面授权并进行报备，绝非可以随意进行的行为。

本节课中我们一起学习了系统安全在蓝队应急响应中的重要性，认识了从手机远控、各类病毒木马到近源渗透等多种攻击手段。蓝队的职责是构建全方位的防御体系，从数字世界到物理世界，守护系统的每一个环节。
