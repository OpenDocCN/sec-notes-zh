# 课程 P84：选举安全 - 保障美国的未来 🗳️🔒

在本节课中，我们将学习美国国土安全部网络安全和基础设施安全局（CISA）在选举安全方面的工作。我们将回顾2016年的经验教训，了解当前为2020年大选所做的准备，并探讨未来确保选举安全的关键措施。

## 2016年的经验教训

上一节我们介绍了课程概述，本节中我们来看看2016年选举安全的情况。大约四年前的这个时间，俄罗斯意图通过多种技术手段干扰2016年大选，这变得非常明显。这些手段包括针对选举基础设施的网络攻击、对政治竞选活动的黑客攻击和泄密，以及更广泛的影响和虚假信息行动。

作为回应，联邦政府动员起来。但当时缺乏现成的专门应对结构和机制。以下是2016年值得注意的三个关键点：

*   **临时性应对**：由于缺乏现有结构，当时的响应是临时拼凑的。这并非错误，只是因为在此前选举安全并非首要任务。
*   **广泛的攻击目标**：俄罗斯人很可能针对了所有50个州，而不仅仅是少数几个摇摆州。这是一场广泛的行动。
*   **未改变选票**：根据现有信息，没有证据表明他们能够通过网络手段影响或改变任何一张选票。这一点至今未变。

为什么当时是临时应对？历史上并没有针对选举安全的专门方案。尽管安全研究社区早已意识到投票设备的风险，但当时没有专门的结构、机制或信息共享与分析中心（ISAC）用于选举安全。相关部门尽己所能，从网络安全角度看，成功防御了2016年的大选。

2016年事件的影响深远。这几乎是一个“斯普特尼克时刻”。在1957年，苏联将斯普特尼克送入近地轨道，这之所以重要，不仅是因为他们先于我们，更因为运载它的实际上是一枚洲际弹道导弹，意味着苏联首次有能力触及美国本土。2016年，通过网络技术，他们再次能够“触及”我们，制造不确定性，并可能破坏民主。此前，网络安全事件多是知识产权盗窃或金融犯罪，但这次的性质截然不同，更加关乎根本。因此，我们知道2018年和2020年的大选面临巨大风险。

## 当前的进展与优势

上一节我们回顾了2016年的挑战，本节中我们来看看当前为加强选举安全所做的努力。与2016年相比，我们现在拥有三个显著优势。

首先，**存在一个活跃的选举安全社区**。自2017年以来，我们帮助培育了这个社区。我们建立了一个信息共享与分析中心（ISAC），如今全美50个州和近7000个司法管辖区都能从中获益，获取威胁信息、最佳实践和战略信息。这是ISAC历史上发展最快的。

其次，**对风险有更好的理解，并增强了风险缓解技术**。我们通过部门协调委员会等机制，汇集各级政府、私营部门和研究人员，共同分析问题、制定最佳实践。我们还利用国会拨款的选举安全补助金，制定了投资指南，指导各州如何最好地降低风险。这些指南基于我们过去几年提供的风险评估、渗透测试等服务，针对官员们的具体挑战量身定制。

第三，**对支持选举管理的系统有更好的可见性**。我们与州和地方选举官员建立了信任关系，这使我们能够与他们合作，提供工具和能力，包括入侵检测系统。截至目前，我们的Albert监控传感器已覆盖所有50个州。在某些州，如佛罗里达州，所有67个县都部署了传感器。这种可见性使我们有信心认为，2020年大选将是现代历史上最受保护、最安全的一次。

## 为2020年及未来做准备

上一节我们探讨了当前的优势，本节中我们来看看为迎接2020年11月大选仍需完成的工作。我们已取得很多进展，但仍有更多事情要做。

我们需要继续为一个**有能力的破坏性对手**做准备。与2016年相比，基于现有的可见性和情报，我们尚未看到类似当年俄罗斯那种有针对性、集中的攻击努力。但我们每天都能看到扫描活动。此外，勒索软件攻击频繁发生，不仅针对州和地方政府，还包括学校、医院等。因此，我们启动了一项选民登记数据库勒索软件倡议，与合作伙伴合作保护这些系统，确保配置正确、启用多因素认证。

保护措施不仅关乎系统和人员，也关乎**流程**。因此，除了加固基础设施，我们还在研究故障转移机制和弹性措施。良好的备份是无价的。我们鼓励在州、县乃至各个投票点层面建立模拟备份。如果选民登记数据库或投票名册出现问题，可以转而使用纸质记录，确保选民能够投票。此外，还有临时选票作为保障。

我们还在全国范围内推出试点项目，引入端点检测与响应能力，将安全防护推向网络边缘，提升整个选举社区的安全水平。

**COVID-19疫情**改变了选举进行的方式。这早在二月份就已明确。我们立即与CDC和HHS合作，通过多年前建立的协调机制，汇集各方力量制定指导和计划。过去几个月，我们通过协调委员会发布了大量指导。改变不仅涉及现场投票，许多州还采用了缺席投票和邮寄投票的方式。我们之前提到纸质选票关联率从80%提升到92%，由于疫情，这个比例可能还会增加。纸质选票对于可审计性至关重要，使我们能够追溯并确定发生了什么，进行有效的选后审计。

最后，2020年大选的一个重要弹性措施将是**知情且有耐心的选民**。计票需要时间，无论是处理缺席选票还是应对更长的队伍。因此，在11月3日，我们很可能不知道谁赢得了选举。我们需要每位选民更有耐心，等待选票得到认证、获胜者确定。这次因为疫情需要更多时间，但这没关系。

同时，我们也需要知情的选民。今年，每个人的投票方式都可能与2016年或2018年不同。可能是新的投票地点，也可能是采用了提前投票、缺席投票或邮寄投票。你需要研究这些变化，制定投票计划，并确保在截止日期前提交信息和选票。最重要的是，保持一点耐心。

## 总结与呼吁

本节课中，我们一起学习了美国CISA在选举安全方面的工作历程。我们从2016年的经验教训出发，了解了当前建立的活跃社区、改进的风险管理和增强的系统可见性等优势。接着，我们探讨了为2020年大选仍需应对的挑战，包括防范有能力的对手、适应COVID-19带来的变化，以及培养知情且有耐心的选民。

请成为解决方案的一部分，加入这个活跃的选举安全社区。我们都在那里捍卫民主。加入CISA，加入州和地方选举社区。因为我们的工作不仅是捍卫今天，更是保障明天。
