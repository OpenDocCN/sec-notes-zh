## 侧信道攻击

**侧信道攻击**是一种间接测量计算结果的方法，通过辅助机制进行。它通常用于攻击加密硬件，因为它攻击的是实现的漏洞，而不是算法本身。

**侧信道攻击的例子**：

* 从硬件比特币钱包中提取比特币钱包的私钥。
* 在安全硬件上进行会话攻击。

**侧信道攻击的漏洞**：

* 软件和硬件之间的交互。
* 算法的实现细节。

1. **收集数据**：使用示波器或其他工具收集目标硬件的信号。
