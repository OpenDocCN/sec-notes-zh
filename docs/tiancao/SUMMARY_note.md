# 天草逆向工程教程目录 📚

在本教程中，我们将系统性地学习天草老师主讲的逆向工程系列课程。课程分为初级、中级、高级三个班次，内容涵盖汇编基础、软件破解、脱壳技术、补丁制作、算法分析以及针对各类商业保护壳的实战对抗。本目录旨在为初学者提供一个清晰的学习路径和内容概览。

## 总结

本节课中，我们一起浏览了天草逆向工程教程的完整课程目录。该体系从**汇编语言**和**基础调试**开始，逐步深入到**软件脱壳**、**补丁制作**、**网络验证对抗**，最终挑战**ASProtect**、**Armadillo**等商业强壳的**算法分析与保护绕过**。建议初学者严格按照初级、中级、高级的顺序进行学习，夯实基础后再挑战更难的内容。每个课程文件都对应一个具体的实战案例或技术点，是逆向工程学习路上宝贵的实践资料。
