# 课程3：《网络安全合规框架与系统管理》：56：1_01_本模块学习内容概述 🎯

![](img/b354348f9b34d1f7abc96aeebebf4509_0.png)

在本模块中，我们将学习组织为保护和防御关键数据而推荐或必须实施的网络安全合规框架与行业标准。我们将探讨相关的法律法规，并深入了解一系列重要的数据保护条例与安全控制措施。

## 法律法规与数据保护条例 📜

上一节我们介绍了本模块的整体学习目标，本节中我们来看看具体的法律法规与数据保护条例。

![](img/b354348f9b34d1f7abc96aeebebf4509_2.png)

首先，我们将探讨相关法律，包括《计算机欺诈与滥用法案》（Computer Fraud and Abuse Act, **CFAA**）。

![](img/b354348f9b34d1f7abc96aeebebf4509_3.png)

![](img/b354348f9b34d1f7abc96aeebebf4509_4.png)

![](img/b354348f9b34d1f7abc96aeebebf4509_5.png)

接下来，IBM团队将引导您了解数据保护法规，包括《通用数据保护条例》（**GDPR**），以及行业特定法规，例如《健康保险流通与责任法案》（**HIPAA**）。

![](img/b354348f9b34d1f7abc96aeebebf4509_6.png)

![](img/b354348f9b34d1f7abc96aeebebf4509_7.png)

![](img/b354348f9b34d1f7abc96aeebebf4509_8.png)

此外，我们还将学习**PCI DSS**（支付卡行业数据安全标准），这是一套针对发行信用卡或处理信用卡支付的组织的信息安全标准。

## 信息安全管理体系标准 🛡️

![](img/b354348f9b34d1f7abc96aeebebf4509_10.png)

![](img/b354348f9b34d1f7abc96aeebebf4509_11.png)

在学习了基础的法律法规后，了解系统化的管理标准同样重要。本节我们将聚焦于信息安全管理体系标准。

**ISO 27000**系列标准有助于组织保障其信息资产的安全。这些标准为建立、实施、维护和持续改进信息安全管理体系提供了框架。

![](img/b354348f9b34d1f7abc96aeebebf4509_13.png)

![](img/b354348f9b34d1f7abc96aeebebf4509_14.png)

## 关键安全控制措施 🔐

![](img/b354348f9b34d1f7abc96aeebebf4509_16.png)

最后，我们将深入学习两套重要的安全控制措施。

以下是本模块将涵盖的两套核心安全控制集：

![](img/b354348f9b34d1f7abc96aeebebf4509_18.png)

*   **CIS关键安全控制**：这是一套推荐的网络防御行动集，提供了具体且可操作的方法来阻止当前最普遍和危险的攻击。
*   **NIST SP 800-53**：该数据库代表了安全控制措施及相关的评估程序，是美国联邦信息系统的重要安全标准。
