上一节我们介绍了课程的整体结构，本节中我们来看看密码学模块的具体学习目标。

以下是本模块将涵盖的核心主题：
*   密码学与加密的基本原理。
*   身份验证与完整性在密码学中的关键作用。
*   密钥分发与认证、哈希函数、公钥基础设施以及证书颁发机构的知识。

接下来，我们将探讨具体的加密概念。

我们将探索诸如**传输中加密**和**静态加密**等加密概念。传输中加密保护正在网络中移动的数据，而静态加密保护存储在设备上的数据。

![](img/0cc1036e9c6c2bb773466f88a84c673e_6.png)

![](img/0cc1036e9c6c2bb773466f88a84c673e_7.png)
