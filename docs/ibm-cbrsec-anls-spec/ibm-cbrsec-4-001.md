# 课程4：《网络安全与数据库漏洞》：1：导论与概述

## 概述

在本节课中，我们将要学习《网络安全与数据库漏洞》的整体框架与核心内容。课程旨在帮助你掌握网络基础原理，并理解关键数据库的漏洞知识。

欢迎来到《网络安全与数据库漏洞》。我是埃里卡·史密斯，IBM身份与访问管理产品组合的课程开发人员。我的职责是帮助你理解每个模块和课程的学习目标，以培养你的网络基础技能，并学习关键术语和数据库漏洞知识。

每一节课都由IBM的安全专家讲授。这些专家不仅负责培训我们的客户，也承担着保护IBM自身全球系统和资源的责任。你将有机会向来自哥斯达黎加、美国和加拿大的领域专家学习。

## 课程核心内容

以下是本课程将涵盖的主要知识点。

*   **网络基础**：你将学习TCP/IP和OSI模型、DNS、DHCP，以及交换与路由的基本概念。
*   **网络技术深入**：将涵盖IP寻址、网络地址转换（NAT）以及数据包嗅探技术。
*   **数据库与漏洞**：你将学习关键数据库的结构与漏洞，包括SQL、CouchDB、Oracle和MongoDB。

这些课程内容的设计覆盖了网络与数据库的基础知识，旨在让你能够充满信心地进入网络安全职场。
