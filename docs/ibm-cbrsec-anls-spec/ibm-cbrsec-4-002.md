# 课程4：《网络安全与数据库漏洞》：2：TCP/IP框架简介 🌐

![](img/5254cc962877efea8e6292daf97a6507_0.png)

在本节课中，我们将学习TCP/IP协议框架的基础知识。TCP/IP是互联网通信的核心，理解其结构对于分析网络安全至关重要。我们将介绍该框架的五层模型，并学习一些关键术语和概念，为后续课程打下坚实基础。

![](img/5254cc962877efea8e6292daf97a6507_2.png)

## 课程概述

![](img/5254cc962877efea8e6292daf97a6507_3.png)

首先，来自IBM哥斯达黎加安全运营中心的安全信息与事件管理管理员Moises Manhe Marin将为我们讲解课程内容。

Moises将引导我们了解TCP/IP协议框架的五层结构，并介绍多个术语和概念。这些知识将为我们在本课程后续的学习奠定基础。

![](img/5254cc962877efea8e6292daf97a6507_5.png)

接下来，我们将学习网络协议，并通过多个概念演示来巩固第一课所学信息。这些演示将帮助我们更直观地理解抽象的网络概念。
