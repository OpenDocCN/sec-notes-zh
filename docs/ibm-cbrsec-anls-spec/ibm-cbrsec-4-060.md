# 课程4：《网络安全与数据库漏洞》：60：TCP/IP框架简介 🖧

在本节课中，我们将学习TCP/IP协议框架的基础知识。TCP/IP是互联网通信的核心，理解其结构对于分析网络安全至关重要。我们将从框架的五层模型开始，为后续课程打下坚实的理论基础。



上一节概述了本课程的学习目标，本节中我们将认识本模块的讲师并了解主要内容。



来自IBM哥斯达黎加安全运营中心的安全信息与事件管理管理员Moises Manhe Marin将在本模块中为大家授课。
