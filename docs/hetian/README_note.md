## 什么是计算机网络？ 🤔

计算机网络是指将地理位置不同的、具有独立功能的多台计算机及其外部设备，通过通信线路连接起来，在网络操作系统、网络管理软件及网络通信协议的管理和协调下，实现资源共享和信息传递的系统。

上一节我们介绍了网络的基本定义，本节中我们来看看网络的核心组成部分。

## 网络的核心组件

一个典型的计算机网络由以下几个关键部分组成：

以下是网络的主要组成部分列表：

*   **节点**：网络中的任何设备，如计算机、服务器、打印机、路由器等。
*   **链路**：连接节点的物理或逻辑通道，如网线、光纤、无线电波。
*   **协议**：一套规则和标准，确保节点之间能够有效通信。例如 **TCP/IP** 协议族。
*   **网络拓扑**：描述网络中节点和链路的物理或逻辑布局结构，如星型、总线型、环型。

## 基础网络模型：OSI与TCP/IP

为了理解数据如何在网络中传输，我们需要参考分层模型。最著名的两个模型是OSI七层模型和TCP/IP四层模型。

**OSI模型** 是一个理论框架，将网络通信过程分为七层：
1.  物理层
2.  数据链路层
3.  网络层
4.  传输层
5.  会话层
6.  表示层
7.  应用层

**TCP/IP模型** 是互联网实际使用的协议栈，分为四层：
1.  网络接口层
2.  网际层（核心协议：**IP**）
3.  传输层（核心协议：**TCP**, **UDP**）
4.  应用层（包含HTTP, FTP, DNS等协议）

一个简单的TCP连接建立（三次握手）可以用伪代码表示其核心思想：
```
客户端发送：SYN
服务器回复：SYN-ACK
客户端确认：ACK
```
连接至此建立。

## 什么是网络安全？ 🛡️

网络安全是指保护网络系统中的硬件、软件及其数据不受偶然或恶意的破坏、更改、泄露，确保系统连续、可靠、正常地运行，网络服务不中断。

从网络基础过渡到安全领域，其核心目标是保障 **CIA 三元组**：
*   **机密性**：防止信息泄露给未授权的个体。
*   **完整性**：防止信息被未经授权的方式篡改。
*   **可用性**：确保授权用户在需要时可以访问信息和资源。

## 常见的网络威胁类型

了解威胁是防御的第一步。网络威胁多种多样，以下是一些常见类型：

以下是初学者需要了解的主要网络威胁列表：

*   **恶意软件**：包括病毒、蠕虫、特洛伊木马、勒索软件等有害软件。
*   **网络钓鱼**：通过伪造的通信（如电子邮件、网站）诱骗用户泄露敏感信息。
*   **拒绝服务攻击**：通过海量流量淹没目标系统，使其无法提供正常服务。
*   **中间人攻击**：攻击者秘密介入两个通信方之间，拦截或篡改通信数据。
*   **未授权访问**：攻击者利用漏洞获取系统或数据的访问权限。

## 基础安全措施

面对威胁，我们可以采取一些基础措施来提升安全性。

以下是每个人都可以实施的基础安全实践列表：

*   **使用强密码**：密码应足够长，混合大小写字母、数字和符号。例如，使用密码管理器生成和保存。
*   **定期更新软件**：及时为操作系统、应用程序安装安全补丁。
*   **安装防病毒软件**：并保持其病毒库为最新状态。
*   **警惕网络钓鱼**：不轻易点击不明链接或打开可疑附件，核实发送方身份。
*   **启用防火墙**：使用硬件或软件防火墙监控和控制网络流量。
