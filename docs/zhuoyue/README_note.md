# 卓越网络安全教程笔记 第1课：网络安全基础 🛡️

在本节课中，我们将要学习网络安全的核心基础概念。这些概念是理解后续所有高级主题的基石，包括威胁、防御措施以及安全模型的基本原理。

## 什么是网络安全？

网络安全是指保护网络、设备、程序和数据免受攻击、破坏或未经授权访问的实践。其目标是确保信息的**机密性**、**完整性**和**可用性**，这通常被称为 **CIA三要素**。

上一节我们介绍了网络安全的定义，本节中我们来看看其核心目标。

## 核心安全目标：CIA三要素

CIA三要素是信息安全的基础模型，它定义了保护的三个主要方面。

以下是CIA三要素的具体内容：

*   **机密性**：确保信息不被未授权的个人、实体或过程访问或泄露。
    *   **实现方式示例**：`加密算法`，如AES。
*   **完整性**：保护信息免受未经授权的篡改，确保其准确和可靠。
    *   **实现方式示例**：`哈希函数`，如SHA-256，用于生成数据“指纹”。
*   **可用性**：确保授权用户在需要时可以可靠地访问信息和资源。
    *   **实现方式示例**：防御`拒绝服务攻击`。

理解了安全目标后，我们需要认识威胁这些目标的对手。

## 常见的网络威胁类型

网络威胁多种多样，了解它们是制定防御策略的第一步。

以下是几种主要的威胁类型：

*   **恶意软件**：旨在破坏、窃取或进行其他有害行为的软件。例如：病毒、蠕虫、勒索软件。
*   **网络钓鱼**：通过伪装成可信实体，诱骗用户泄露敏感信息（如密码、信用卡号）的攻击。
*   **拒绝服务攻击**：通过洪水般的流量使目标系统资源耗尽，导致合法用户无法获得服务。
*   **中间人攻击**：攻击者秘密拦截并可能篡改双方之间的通信。

认识了威胁，接下来我们探讨如何构建防御体系。

## 基础防御措施

有效的网络安全依赖于分层防御策略，即“纵深防御”。

以下是关键的基础防御措施：

*   **防火墙**：作为网络边界的安全屏障，根据预定规则控制进出网络的数据流。
    *   **核心功能**：`过滤数据包`。
*   **防病毒/反恶意软件**：用于检测、阻止和清除恶意软件的程序。
*   **强密码与多因素认证**：使用复杂密码并结合多种验证方式（如手机验证码）来保护账户。
*   **软件更新与补丁管理**：及时修复软件漏洞是防止被利用的关键。
*   **数据加密**：将数据转换为密文，即使被截获也无法直接读取。
    *   **核心概念**：`明文 --[加密密钥]--> 密文`。
