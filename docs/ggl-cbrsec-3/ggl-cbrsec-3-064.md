# 064：恶意数据包嗅探

![](img/64979f157b8987a9c803a329e9659b5d_0.png)

在本节中，我们将学习数据包嗅探技术，并重点探讨威胁行为者如何利用此技术非法获取信息。我们将了解数据包嗅探的原理、其恶意应用形式以及如何防御此类攻击。

## 数据包与数据包嗅探

上一节我们介绍了在网络中传输的数据包所包含的信息。本节中我们来看看威胁行为者如何利用这些信息。

数据包由**包头**和**包体**组成。包头包含发送方和接收方的IP地址。包体则可能包含有价值的信息，例如姓名、出生日期、个人消息、财务信息或信用卡号。

**数据包嗅探**是指使用软件工具观察网络传输中数据的过程。作为安全分析师，你可能会在调查事件或调试网络问题时使用数据包嗅探来分析捕获数据包。在本证书课程的后续部分，你将有机会亲手实践一些数据包嗅探软件。

然而，恶意行为者也可能使用数据包嗅探来查看并非发送给他们的数据。这有点像私自拆看他人的信件。了解威胁行为者如何怀有恶意地使用数据包嗅探至关重要，这样你才能做好准备，防范这些恶意行为。

## 恶意数据包嗅探的工作原理

恶意行为者可能会将自己插入两个设备之间的合法连接中间。然后，他们就可以使用数据包嗅探来监视流经其设备的每一个数据包。其目的是在数据包中找到有价值的信息，以便加以利用。

攻击者可以使用软件应用程序或硬件设备来查看数据包。恶意行为者可以通过数据包嗅探器访问网络数据包，并修改其中的数据。他们可能修改包体中的信息，例如更改收款人的银行账号。

数据包嗅探可分为被动式和主动式。

## 被动式数据包嗅探

被动式数据包嗅探是一种在数据传输过程中读取数据包的攻击类型。

由于集线器上所有主机都能看到网络上的所有流量，恶意行为者可以查看进出其目标设备的所有信息。

回想一下信件投递的例子，我们可以将被动式数据包嗅探攻击比作邮递员恶意地阅读他人信件。邮递员（或数据包嗅探器）有权投递邮件，但无权阅读其中的信息。

## 主动式数据包嗅探

主动式数据包嗅探是一种在数据传输过程中操纵数据包的攻击类型。

这可能包括注入互联网协议以将数据包重定向到非预期的端口，或更改数据包包含的信息。

主动式数据包嗅探攻击就像邻居告诉邮递员：“我来帮你投递那封信”，然后在把信放入你的邮箱之前阅读或更改信件内容。即使你的邻居认识你，并且将信送到了正确的房子，他们也是在主动地、刻意地进行恶意行为。

好消息是，恶意数据包嗅探是可以预防的。

## 如何防御恶意数据包嗅探

以下介绍网络安全专业人员可以采取的几种防御此类攻击的方法。

一种防御恶意数据包嗅探的方法是使用**VPN**来加密和保护在网络中传输的数据。如果你不记得VPN的工作原理，可以回顾本课程前一节中关于此主题的视频。当你使用VPN时，黑客可能会干扰你的流量，但他们无法解码并读取你的私人信息。

另一种增加防御层的方法是确保你访问的网站在域名地址开头使用**HTTPS**。之前我们讨论过，HTTPS使用SSL/TLS来加密数据，防止恶意行为者在监视网络传输时进行窃听。

最后一种帮助保护自己免受恶意数据包嗅探的方法是避免使用**未受保护的Wi-Fi**。你通常可以在咖啡馆、餐厅或机场等公共场所找到未受保护的Wi-Fi。这些网络不使用加密。这意味着网络上的任何人都可以访问进出你设备的所有数据。你可以采取的一项预防措施是避免使用免费的公共Wi-Fi，除非你的设备上已经安装了VPN服务。
