## 下载

### Docker

```
docker pull apachecn0/sec-notes-zh

# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g sec-notes-zh
sec-notes-zh <port>
# 访问 http://localhost:{port} 查看文档
```
